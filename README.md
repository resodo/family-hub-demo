# Family Hub Demo

一个用 AI（Claude Code + Claude App）管理家庭健康与日常生活的真实案例。

## 这是什么？

这是一个 **markdown 文件 + AI + 静态网站** 组成的家庭管理系统。核心理念：

> **不需要开发 app，不需要后端。一堆 markdown 文件 + git + AI = 家庭操作系统。**

### 虚构但真实

项目中的人物和数据是虚构的（保护隐私），但**系统设计、使用方式、和 AI 的协作模式完全真实**——这就是我每天在用的东西。

## 系统架构

```
┌─────────────────────────────────────────────────────────┐
│                      数据层（Markdown + Git）              │
│  health/*.md  recipes/*.md  todo.md  private/*.md        │
│                    ↕ 版本管理 (git)                       │
├─────────────────────────────────────────────────────────┤
│                      AI 交互层                            │
│  ┌──────────────┐    ┌──────────────┐                    │
│  │  Claude Code  │    │  Claude App  │                    │
│  │  (桌面深度交互) │    │  (移动端轻量)  │                    │
│  │  排菜、更新档案  │    │  快速查询、提醒 │                    │
│  └──────────────┘    └──────────────┘                    │
│         ↕ MCP                  ↕                         │
│  Google Calendar / 小红书 / ...                           │
├─────────────────────────────────────────────────────────┤
│                      展示层（VitePress 静态网站）           │
│  家人通过加密网站查看最新内容                                │
│  push → GitHub Actions 自动构建 → 部署                    │
└─────────────────────────────────────────────────────────┘
```

## 核心场景

### 1. 家庭健康管理
- 孕期全程跟踪（产检记录、用药安全、营养补剂）
- 幼儿健康管理（过敏脱敏、生长发育、疫苗接种）
- 成人健康档案（体检追踪、慢病管理）

### 2. 每日餐饮规划
- 和 AI 对话式排菜，自动检查孕期食品安全 + 儿童过敏原
- 输出阿姨可执行的微信指令（纯文本，直接复制发送）
- 菜谱库持续积累，避免重复

### 3. 生活事务管理
- 待办事项（产检预约、体检、家庭规划）
- Google Calendar 集成（通过 MCP）
- 家庭决策记录

## 为什么用这种方式？

### 低摩擦是关键
我不需要打开任何 app、填任何表单。我只需要**和 AI 说话**：
- "糖豆今天吃了半个蛋黄没反应，记一下"
- "安排明天的饭"
- "小鱼的唐筛结果出了，低风险"

AI 自动把信息写入正确的文件、正确的位置。

### CLAUDE.md = Prompt Engineering
`CLAUDE.md` 是整个系统的"操作手册"。AI 每次对话都会自动加载，确保：
- 知道家庭成员是谁，各自的健康状况
- 知道排菜规则（孕期安全、过敏原检查）
- 知道文件结构，修改时保持跨文件一致性

### Markdown + Git = 免费的数据库
- 每次修改自动 diff，完整历史可追溯
- 不依赖任何第三方服务
- 文件即数据，结构即 schema

## 如何使用

### 本地预览网站
```bash
npm install
npm run dev
```

### 搭配 Claude Code 使用
1. 在项目根目录打开 Claude Code
2. AI 自动读取 `CLAUDE.md`，了解所有规则
3. 开始对话即可

### 搭配 Claude App (移动端) 使用
1. 创建 Project，上传 `CLAUDE.md` 和关键文件
2. 适合外出时快速查询和记录
3. 回到电脑后同步更新到 git
