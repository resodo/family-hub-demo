# 蔬菜、蛋豆制品、汤菜谱

### 虾仁蒸蛋

食材（2人份）：
- 鸡蛋 3个（约150g）
- 温水 255ml（蛋液的1.7倍）
- 虾仁 8只（约100-120g）
- 盐 1.5g（蛋液用）
- 白胡椒粉 少许（腌虾）
- 葱花 少许
- 蒸鱼豉油 5ml（可选）

1. 鸡蛋打散，加盐和温水搅匀，过筛1-2次，静置2分钟去泡沫。
2. 虾仁加少许盐和白胡椒，抓匀腌10分钟。
3. 蛋液碗盖盘（或保鲜膜扎孔），水开后中小火蒸10分钟。
4. 打开盖子摆上虾仁，再蒸8-10分钟至虾仁全熟、蛋液全凝固。
5. 撒葱花，按需淋5ml蒸鱼豉油。

关键技巧：
- 蛋液和水建议1:1.7~2，越高越嫩但更难定型。
- 孕期版必须全熟：虾仁不透明、蛋液无流心。

### 清炒豌豆苗

食材（2人份）：
- 豌豆苗 300g（挑嫩的，老茎嚼不动）
- 蒜 3-4瓣（约10g，切末）
- 橄榄油 15ml（1汤匙）
- 盐 2g（约1/3茶匙）

做法：
1. 豌豆苗摘去老茎和黄叶，只留嫩梢和嫩叶，清水洗净沥干（⚠️ 一定挑嫩的，老茎炒完嚼不烂）
2. 热锅倒油15ml，**大火**烧至油面微微冒烟
3. 放入蒜末快速翻炒5秒爆出香味（大火不怕蒜焦，因为马上就要下菜了）
4. 立即倒入全部豌豆苗，**大火快炒**，用锅铲不停翻动，让每片叶子都沾上油
5. 炒至豌豆苗**刚刚变软塌下去**（约30-40秒），立刻加盐2g，再翻炒均匀**5秒**
6. **立即出锅**，不要犹豫。从下菜到出锅，全程不超过1分钟

关键技巧与避坑：
- **全程大火+快速**是唯一要诀，炒久了会出水变烂+发苦发黑
- 下锅前沥干水分，带水下锅=煮菜不是炒菜
- 盐要最后放，提前放会逼出水分
- 出锅要果断，余热还会继续加热，宁欠勿过
- 豌豆苗缩水率很高（约60%），300g生的炒完只有一小盘，别嫌少

### 紫菜豆腐汤

食材（2-3人份）：
- 嫩豆腐 300g（切1.5cm小块）
- 紫菜 5g（半片）
- 虾皮 8-10g
- 蒜末 1瓣
- 葱白 1根
- 清水 650ml
- 生抽 5ml
- 盐 1.5g
- 白胡椒粉 少许
- 葱花 少许

1. 锅内少油，下蒜末、葱白、虾皮爆香20秒。
2. 加清水和豆腐，煮开后中小火2-3分钟。
3. 下紫菜再煮1-2分钟。
4. 加生抽、盐和白胡椒调味，撒葱花出锅。

关键技巧：
- 紫菜最后放，避免煮太久发烂、汤发浑。
- 豆腐下锅后轻推，别猛搅防碎。

### 冬瓜口蘑汤

食材（2-3人份）：
- 冬瓜 350g（去皮去瓤，切薄片）
- 口蘑 200g（切片）
- 姜丝 5g
- 蒜末 5g
- 葱花 少许
- 清水 700ml
- 生抽 5ml
- 蚝油 3ml（可选，提鲜）
- 盐 1.5g
- 白胡椒粉 0.5g
- 食用油 8ml

1. 冬瓜去皮去瓤切片，口蘑切片，姜切丝，蒜切末。
2. 锅中下油，先下姜丝和蒜末炒香10秒。
3. 下口蘑中火翻炒2-3分钟，炒到口蘑明显出汁、香味出来。
4. 下冬瓜翻炒30秒，倒入清水700ml，大火煮开后转中小火。
5. 加生抽、（可选）蚝油，继续煮10-12分钟，至冬瓜半透明软熟。
6. 最后加盐和白胡椒调味，撒葱花出锅。

关键技巧：
- 先炒口蘑出汁是这道汤鲜味的关键。
- 冬瓜煮到半透明就可以，不要煮烂成泥。
- 这道汤不需要糖。

### 红薯

**烤红薯**（烤箱/空气炸锅）：
- 洗净，两头切掉一小块（利于导热，避免两头有丝）
- 200度 55-60分钟（根据大小调整），不包锡纸
- 想流油：先烤20分钟→拿出蘸一下清水→继续烤30-40分钟
- 趁热吃更香甜

**蒸红薯**（隔水蒸）：
- 先烧开水，再放入红薯，大火蒸40分钟
- 冷透后吃更香甜

### 豆豉鲮鱼油麦菜

食材：油麦菜、豆豉鲮鱼罐头（甘竹牌）、大蒜

备料：
- 鲮鱼罐头打开，鱼肉和骨头拆分，骨头扔掉，鱼肉手撕成块
- 罐头里的油和豆豉不要丢
- 大蒜切末，分两份
- 油麦菜根和叶分开

1. 罐头里的油全部倒进锅，豆豉只放1/3（全放太咸），小火煸香
2. 下一半蒜末煸香，再下鲮鱼块翻炒
3. 下油麦菜根，炒至微软
4. 最后放另一半蒜末和油麦菜叶，翻炒1-2分钟出锅

不需要再放任何调味料，包括盐

### 蛋黄茄汁煎豆腐

食材：
- 北豆腐1盒（不要用卤水豆腐/柴火豆腐，卤味太重会冲突；韧豆腐也可以，见下方变体）
- 咸鸭蛋3-4个取蛋黄（新鲜剥的比现成咸蛋黄好，蛋白可以佐粥）
- 罐装番茄1/3~1/2罐（推荐意大利产去皮番茄罐头或新疆「冠农」品牌，不要用新鲜番茄或番茄酱——鲜番茄味道太寡淡，要更多量+更长时间才能达到效果）
- 高汤1碗（鸡汤/牛肉汤/排骨汤都行）
- 小葱2-3根
- 盐1茶匙

**1. 熬汤底**
- 锅里1瓷勺油，中火烧热转小火，放咸蛋黄炒到起泡
- 尽量用锅铲把蛋黄碾碎再碾碎，要全变成沙质口感没有颗粒（容易炒糊，可以时不时关火用余温操作）
- 加番茄罐头，小火炒2分钟，让番茄和蛋黄彻底融合
- 加高汤和盐，小火煮1分钟，煮匀备用

**2. 煎豆腐**
- 另一口不粘锅，约1.5瓷勺油，中火烧热（豆腐需要油但又不吃油，油量不要大）
- 豆腐切1.5cm厚片，能用厨房纸吸一下表面水分更好（不容易溅油）
- 均匀码在锅里，中小火每面煎2分钟至焦黄，动作轻柔保持完整

**3. 煮豆腐**
- 倒入备好的汤底，没过豆腐（不够的话加高汤或清水补）
- **先尝汤底咸淡**，刚好或偏咸1分都可以，调好了再煮
- 煮沸后转小火继续煮5分钟（北豆腐一定要煮够时间，味道才能进去）
- 撒葱花出锅

**韧豆腐变体：**
- 切1.5cm方块，用锅铲朝里推，不要来回拨，豆腐不容易碎
- 韧豆腐气孔紧密不易入味，需要勾芡让汤汁挂上去
- 1茶匙淀粉+20-30ml清水，入锅前再次搅匀
- 汤汁煮到食材高度一半左右时，淀粉水分2-3次倒入（这样芡收得紧不易泄），每次倒入后迅速搅拌均匀

### 清炒荷兰豆（蒜香版）

食材：
- 荷兰豆 300g（掐两头撕两侧老筋）
- 大蒜 4-5瓣（约20g，一半切蒜片一半切蒜蓉）
- 食用油 15ml（1汤匙）
- 生抽 15ml（1汤匙）
- 蚝油 10ml（约2/3汤匙）
- 盐 2g（1/3茶匙）
- 0卡糖 一小撮

焯水用：一锅水 + 盐2g + 几滴油

1. 荷兰豆掐两头撕两侧老筋（不能省，否则口感柴），洗净沥干
2. **焯水**：烧一锅水开后加盐2g+几滴油（保色），倒入荷兰豆焯水**20-30秒（不超过30秒）**，立即捞出**过凉水**，沥干
3. 锅烧热倒油，中大火先下**蒜片**爆香10-15秒（微金黄即可）
4. 倒入沥干的荷兰豆**大火快炒30秒**
5. 加生抽1汤匙+蚝油2/3汤匙+盐1/3茶匙+0卡糖一小撮+**蒜蓉**，大火翻炒30秒出锅

**全程炒制不超过1.5分钟。**

关键技巧：
- 焯水加盐加油保色，**不超过30秒+过凉水**=脆绿不变色
- **沥干水分再下锅**，带水下锅降温容易出水变蔫
- **蒜分两次下**：蒜片先下爆香出底味，蒜蓉最后下保留蒜香
- 荷兰豆已焯过水是半熟状态，炒太久就软了

### 蒜蓉西兰花

食材（2-3人份）：
- 西兰花1棵（约350-400g），掰均匀小朵
- 大蒜6-8瓣（约20g），剁蒜末，分两份（2/3先炒 + 1/3出锅撒）
- 食用油2汤匙（约30ml）

酱汁（提前调好）：
- 蚝油1.5汤匙（15ml）
- 生抽1汤匙（10ml）
- 白糖1/2茶匙（3g）
- 清水3汤匙（45ml）
- 水淀粉：1茶匙淀粉 + 2汤匙水

1. 西兰花淡盐水泡10分钟洗净
2. 焯水：大锅水烧开，加少许盐+几滴油，西兰花下锅30秒-1分钟（变深绿立刻捞），马上过凉水，控干水分（轻轻挤压）
3. 炒锅大火烧热下油，五成热下2/3蒜末爆香（15秒，微变色不能焦）
4. 倒入调好的酱汁，大火烧开冒泡
5. 下西兰花快速翻炒裹汁（30秒-1分钟）
6. 淋水淀粉翻炒10秒，酱汁浓稠挂住
7. 关火，撒剩余1/3生蒜末，翻几下出锅

关键技巧：
- 蒜末分两次放：熟蒜焦香+生蒜辛香叠加，比一次全放香得多
- 焯水严格30秒-1分钟，超时变软发面口感全毁
- 焯水后必须过凉水，保持翠绿不发黄
- 控干水分再下锅，不然酱汁挂不住、成品水汪汪
- 勾薄芡让酱汁挂在西兰花上而不是流到盘底
- 全程大火快炒，锅里只要30秒裹酱即可

参考来源：[下厨房](https://m.xiachufang.com/recipe/26334/)、[美食天下](https://home.meishichina.com/recipe-223407.html)、[香哈网](https://m.xiangha.com/caipu/76864491.html)

### 上汤白菜

食材（2人份）：
- 大白菜 半棵（取菜心嫩叶部分，约300g）
- 咸鸭蛋 1个（蛋黄蛋白分开，各切小块）
- 蒜 5-6瓣（整粒不切）
- 虾皮 1小把（约10g，提鲜）
- 枸杞 少许（温水泡软，出锅撒）
- 高汤 1大碗（300-400ml，冻高汤块+开水化开）
- 食用油 少许

调味：
- 白胡椒粉 少许
- 盐 视情况（咸鸭蛋+高汤本身有咸味，先尝再决定加不加）

1. 白菜洗净，纵向劈成条，切约10cm长段
2. 锅倒少许油，蒜粒中火炸到金黄，捞出备用
3. 锅里余油，**先下咸蛋黄小火炒到起泡冒沙**（约半分钟），再下咸蛋白和虾皮翻炒几下
4. 倒入高汤（冻高汤块+开水），放回炸好的蒜粒，大火烧开
5. 放入白菜，中火煮3-5分钟至白菜软但不烂
6. 尝咸淡，不够补一点盐，撒白胡椒粉，出锅撒枸杞

关键技巧：
- 咸蛋黄先炒出沙是灵魂，能让汤底变得浓郁鲜香
- 咸鸭蛋本身有盐，盐一定最后尝了再决定加不加
- ~~皮蛋版~~：❌ 孕期禁用皮蛋（含铅，详见食材红线）

参考来源：[下厨房（上汤白菜）](https://m.xiachufang.com/recipe/1074872/)、[美食天下（上汤大白菜）](https://home.meishichina.com/recipe-242532.html)

### 白菜炖豆腐

食材（2-3人份）：
- 大白菜 400-500g（帮斜片薄片，叶手撕，帮和叶分开）
- 北豆腐 300g（切2cm方块）
- 虾皮/海米 15-20g（海米需温水泡10分钟，**泡的水留着**）
- 葱2根切花，姜1小块切末

调味：
- 盐 3/4茶匙（3-4g，虾皮有咸味所以盐少放）
- 白胡椒粉 1/4茶匙（1g）
- 料酒 2茶匙（10ml）
- 食用油 15ml（炒菜用）
- 香油 1茶匙（5ml，出锅淋）

1. 豆腐焯水2-3分钟（去豆腥、更紧实不易碎），捞出沥干
2. 锅热油15ml，**先炒白菜帮**1-2分钟→再放白菜叶炒软，盛出
3. 同锅补少许油，中小火煸虾皮/海米+葱姜30秒出香，烹入料酒
4. 倒入泡海米的水+清水共300-400ml（白菜会出水，别加太多），大火烧开
5. 放入白菜和豆腐，加盐和白胡椒粉，转小火**盖盖炖15-20分钟**（千滚豆腐万滚鱼，久炖才入味）
6. 尝咸淡，淋香油出锅

关键技巧：
- 白菜帮和叶分开处理，帮先炒再炖才不会有青草味
- 虾皮/海米是鲜味来源，不需要加酱油蚝油
- 豆腐先焯水才紧实不易碎
- 炖够15-20分钟，豆腐才能入味

参考来源：[百度百科](https://baike.baidu.com/item/%E7%99%BD%E8%8F%9C%E7%82%96%E8%B1%86%E8%85%90/2569033)、[下厨房](https://m.xiachufang.com/recipe/100623769/)、[知乎](https://zhuanlan.zhihu.com/p/87823023)

### 清炒丝瓜

食材（2-3人份）：
- 丝瓜 2根（约500g）
- 大蒜 4瓣（约15g），切末
- 橄榄油 2汤匙（约25ml，比一般炒菜略多，锁水防黑）
- 盐 1/2茶匙（约2g）

1. 丝瓜去皮，切滚刀块（约2cm）。**切好立即炒，放超过2分钟会氧化变黑**
2. 锅烧热下油，中火煸香蒜末（15秒，微黄即可）
3. **转大火**，下丝瓜快速翻炒约1-1.5分钟，炒到微微变软、边缘半透明
4. **关火后**加盐翻匀，立即出锅装盘

关键技巧：
- **盐必须出锅前放**（最关键！早放盐→丝瓜大量出水→变黑变烂）

### 青椒土豆丝

食材（2-3人份）：
- 土豆 2个（约400g）
- 青椒 2个（切丝）
- 蒜 3瓣（切片）
- 干辣椒 3-4个（剪段，可减量/不放）
- 葱花 少许（出锅撒）
- 食用油 30ml

调味：
- 白醋 20ml（1.5汤匙）— **必须白醋**，黑醋/陈醋会让土豆丝发暗
- 生抽 15ml（1汤匙）
- 蚝油 8ml（半汤匙，代替味精提鲜）
- 盐 3g（半茶匙）

做法：
1. 土豆去皮切2-3mm薄片再切均匀细丝，**立即放清水反复冲洗3-4遍**洗掉淀粉（水变清为止），泡清水备用
2. **焯水（可选但推荐）**：水开下土豆丝**只焯20秒**，迅速捞出**过冰水**冲透，彻底沥干

6. 放青椒丝再翻炒约30秒（保持脆和翠绿）
7. 撒葱花关火出锅

关键技巧：
- **洗掉淀粉**是防粘的根本，不洗必粘成一坨
- **醋要早加**：入锅就加，醋酸让土豆丝保持脆度
- **必须用白醋**，陈醋/黑醋颜色发暗
- **大火快炒不超3分钟**
- 青椒最后30秒放
- 不放老抽（会变黑）

### 蒜蓉粉丝蒸娃娃菜

食材（2人份）：
- 娃娃菜 1颗（约300g）
- 龙口绿豆粉丝 1把（约50g，绿豆粉丝比红薯粉丝更Q弹耐蒸）

- 蚝油 15ml（1汤匙）
- 0卡糖 5g（半汤匙，提鲜去涩）
- 清水 30ml（2汤匙，稀释酱汁使其更均匀渗透）

做法：
1. **粉丝**温水泡10分钟至软（不要用开水泡，开水泡的粉丝蒸完会糊烂），沥干剪成两三段方便夹
2. **娃娃菜**对半切开再切成条，烧开水**焯3-4分钟**捞出沥干（⚠️ 关键步骤！不焯水直接蒸需要20分钟，粉丝会过度软烂；焯过水的蒸8分钟就够，娃娃菜脆嫩粉丝Q弹两全）

6. 将酱汁**均匀淋在粉丝上**（重点淋粉丝，粉丝会吸汁变入味）
7. 锅中水烧开，放入盘子，**大火蒸8-10分钟**
8. 出锅撒葱花

- 生抽+蚝油本身咸度足够，**不额外加盐**，出锅尝味再决定
- 酱汁要**趁热淋**，冷酱渗透力差
- 粉丝用龙口绿豆粉丝最佳，耐蒸不糊

### 酱烧茄子（煎焖版，2-3人份）

主料：
- 茄子（紫皮长茄）2根（约500g），切滚刀块
- 蒜 6瓣（约20g），切末，**分两份**：2/3先炒 + 1/3出锅撒

酱汁（提前调好）：
- 郫县豆瓣酱 1.5汤匙（约20g，颗粒粗的先剁细）
- 生抽 1汤匙（15ml）
- 蚝油 1茶匙（5ml）
- 0卡糖 1茶匙（约5g）
- 料酒 1茶匙（5ml）
- 老抽 几滴（上色，不能多）
- 清水 150ml
- 水淀粉：淀粉1茶匙 + 水2汤匙（收汁用）

其他：
- 食用油 3汤匙（45ml，煎茄子用）
- 香油 1茶匙（5ml，出锅淋）
- 葱花 少许

做法：
1. **腌茄子（省油关键）**：茄子切块后撒1/2茶匙盐，拌匀腌10分钟，用手挤干水分（腌过的茄子不吸油，不腌的像海绵一样疯狂吃油）
2. **煎茄子**：锅中倒3汤匙油，中火烧热，放入茄子块平铺，**不要翻动**，煎约2分钟至底面金黄再翻面，两面煎至金黄微软，盛出备用
3. **炒酱底**：锅中留底油，**中小火**下2/3蒜末爆香（15秒），加豆瓣酱**小火炒1-2分钟至出红油**（⚠️ 必须炒透出红油，否则酱味生涩）
4. **焖烧**：倒回煎好的茄子，加生抽、蚝油、老抽、0卡糖、料酒、清水150ml，大火烧开后**转小火加盖焖5分钟**
5. **收汁**：开盖转大火，淋入水淀粉翻炒至汤汁浓稠挂在茄子上
6. **出锅**：关火，撒剩余1/3生蒜末+葱花翻匀，淋香油出锅

关键技巧：
- **腌制挤水**是省油核心，不腌的茄子会把所有油都吸走
- **豆瓣酱必须小火炒出红油**，炒不透味道发生发涩
- **最后撒生蒜末**是画龙点睛，熟蒜焦香+生蒜辛香双层蒜味
- 不需要额外加盐，豆瓣酱+生抽咸度够了
- 茄子煎的时候不要频繁翻动，耐心等一面定型再翻

### 素炒杏鲍菇（酱汁煎版，2人份）

主料：
- 杏鲍菇 2-3个（选粗的，约300g），切0.5-0.8cm厚片，单面划十字花刀（浅切不切断，帮助入味）

- 蚝油 1汤匙（15ml）
- 0卡糖 1/2汤匙（约5g）
- 淀粉 1汤匙（约10g）
- 清水 80-100ml

辅料：
- 蒜 3瓣，切末
- 葱花 2根
- 食用油 2汤匙（30ml）

做法：
1. **调酱汁**：碗中混合生抽、老抽、蚝油、0卡糖、淀粉、清水，搅匀备用
2. **煎杏鲍菇**：平底锅中火热油，放入杏鲍菇片平铺，**不要急着翻面**，一面煎至金黄（约2-3分钟）再翻面，两面都煎至金黄、水分逼出。盛出备用

- **切厚片（0.5cm以上）**：太薄煎完口感发硬发柴
- **花刀只切一面**：帮助入味但保持完整，不要切断
- **煎的时候要有耐心**：充分煎到两面金黄逼出水分，外焦内嫩全靠这步
- **焖煮是灵魂**：盖盖焖5-6分钟让酱汁渗入菇体，比单纯翻炒入味得多

### 辣炖地三鲜（村驴版，2-3人份）

<!-- 村驴老师（B站约289万粉）版本，不用油炸，酱香辣炖 -->

主料：
- 土豆 1个（约200g），切滚刀块
- 茄子 1.5根（约350g），切滚刀块
- 螺丝椒 2个（约100g），切段

调味：
- 葱 1段，切段
- 姜 3片
- 蒜 4瓣，切末
- 十三香 1/2茶匙
- 0卡糖 1汤匙（约12g）
- 黄豆酱 3汤匙（约50g）
- 辣椒面 2汤匙（约12g，怕辣减量）
- 生抽 2汤匙（约30ml）
- 老抽 少许（上色，几滴即可）
- 蚝油 1.5汤匙（约20ml）
- 食用油 3汤匙（45ml）

做法：
1. **炒酱底（关键步骤）**：锅中倒油烧热，放入葱姜蒜、十三香，炒香。加0卡糖1汤匙、黄豆酱3汤匙、辣椒面2汤匙，**小火慢炒**至出红油、酱香浓郁（约1-2分钟，⚠️ 必须小火，大火酱会糊）
2. **调味**：加生抽2汤匙、老抽少许，继续小火熬出香味，等酱料冒泡

4. **加水炖煮**：加水没过食材大半（约300-400ml），沿锅边淋蚝油，大火煮开
5. **小火慢炖30分钟**：转小火加盖炖30分钟，中途翻动1-2次防粘底
6. **收汁**：开盖转大火收汁至浓稠即出锅

关键技巧：
- **不用油炸**是这个版本的核心，比传统地三鲜省油省事
- **酱底必须小火炒透出红油**，黄豆酱+辣椒面+十三香的酱底是灵魂
- **小火慢炖30分钟**让食材充分入味，土豆软糯、茄子吸满酱汁
- 辣椒面用量按口味调整，2汤匙是中辣，怕辣减至1汤匙
- 村驴老师视频中用"勺"做量词，以上克数是根据视频换算的参考值，可按口味微调

### 家常豆腐（2-3人份）

<!-- 去掉木耳版本 -->

主料：
- 北豆腐（老豆腐）1块（约350-400g），切2cm厚三角片或方块
- 猪肉末 50g（可选，增香）

配菜：
- 青椒 1个（约80g），切片
- 胡萝卜 半根（约60g），切薄片
- 葱 1段（约15g），切段/葱花
- 姜 3-4片（约10g）
- 蒜 3-4瓣（约10g），切末

调味：
- 郫县豆瓣酱 1.5汤匙（约20g，⚠️ 必须郫县的，颗粒粗先剁细）
- 生抽 1茶匙（5ml）
- 老抽 2-3滴（上色，不可多）
- 0卡糖 1/2茶匙（约2g）
- 料酒 1茶匙（5ml，腌肉末用）
- 清水 200ml
- 水淀粉：淀粉1汤匙 + 水2汤匙
- 食用油：煎豆腐3汤匙（45ml）+ 炒菜1汤匙（15ml）
- **不额外加盐**（豆瓣酱+生抽+老抽咸度已够）

做法：
1. **豆腐处理**：豆腐切块，放入淡盐水（500ml水+1/2茶匙盐）浸泡15-20分钟（去豆腥、更结实不易碎），捞出用厨房纸吸干水分
2. **煎豆腐**：锅中倒3汤匙油，中小火烧热，豆腐逐块放入，**不要急着翻面**，煎2-3分钟至底面金黄定型再翻面，两面金黄后盛出
3. **炒底料**：锅中留1汤匙底油，中火下姜片、蒜末爆香30秒。如有肉末此时下锅炒散变色
4. **炒豆瓣酱（关键）**：转**中小火**，下1.5汤匙郫县豆瓣酱，**炒1-2分钟至出红油、香味浓郁**（⚠️ 豆瓣酱必须炒透出红油）
5. **下配菜**：转中火，先下胡萝卜翻炒1分钟，再加青椒翻炒30秒
6. **烧豆腐**：放入煎好的豆腐轻轻翻匀，加生抽、老抽、0卡糖，倒入清水200ml，大火烧开后转**中小火加盖焖3-5分钟**入味
7. **收汁出锅**：开盖倒入水淀粉，转大火翻炒30秒至汤汁浓稠裹住豆腐，撒葱花出锅

关键技巧：
- **必须用北豆腐/老豆腐**，嫩豆腐一煎就碎
- **郫县豆瓣酱是灵魂**，必须小火炒出红油，炒不透味道发生发涩
- **煎豆腐要耐心**：放入后不动，等底面完全金黄定型再翻，否则粘锅碎裂
- **不加盐**：豆瓣酱+生抽+老抽提供足够咸度
- 淡盐水泡豆腐是防碎+去腥的好方法

### 凉拌藕片（2-3人份）

主料：
- 莲藕 1-2节（约350-400g，选藕尖部位更脆嫩）

调料：
- 大蒜 3瓣（约15g），切末
- 小米辣 2-3个，切圈（怕辣减量）
- 干红辣椒 3-4个，切斜段
- 花椒 一小撮（约2g）
- 香醋/陈醋 2汤匙（30ml）
- 生抽 2汤匙（30ml）
- 0卡糖 1汤匙（约10g）
- 盐 1/2茶匙（约3g）
- 白芝麻 10g
- 香油（芝麻油）1汤匙（15ml）
- 辣椒油 1茶匙（5ml）
- 食用油 2汤匙（30ml，泼油用）
- 香菜 1-2根，切段（可选）

做法：
1. **切藕片**：莲藕洗净去皮，切约2mm薄片（尽可能薄，更脆更入味），切好泡清水防氧化
2. **焯水**：锅中烧开水，加几滴白醋（帮助保持洁白），放入藕片焯**1-2分钟**（⚠️ 不超过3分钟，否则失去脆感）
3. **过冰水**：捞出藕片**立即放入冰水/凉水**浸泡冷却（⚠️ 关键步骤！热藕片会继续焖熟，过凉水锁住脆度），彻底凉透后沥干水分
4. **准备调料碗**：蒜末、小米辣圈、干红辣椒段、白芝麻放入碗中
5. **泼热油**：炒锅倒2汤匙食用油，放花椒粒小火炸香（约30秒），捞出花椒不要，趁热将花椒油浇到调料碗中激发香味
6. **调酱汁**：碗中加香醋（先放醋有助保脆）、生抽、0卡糖、盐、香油、辣椒油，搅匀至糖融化
7. **拌匀**：沥干的藕片放入大碗，倒入酱汁，加香菜段，充分拌匀让每片藕都裹上酱汁
8. 可立即吃，也可冷藏15-30分钟更入味

关键技巧：
- **切薄**：2mm是最佳厚度，太厚不入味、口感发面
- **焯水不超过2分钟**：宁欠勿过，焯太久发软发面
- **必须过冰水/凉水**：锁住脆度的关键
- **先放醋再放其他调料**：醋有助保持藕片脆爽
- **充分沥干**：藕片带太多水会稀释酱汁
- 选藕尖部位（靠近两端较细的部分），口感更脆嫩

### 孜然小土豆（干煸版，2-3人份）

主料：
- 小土豆 500g（约1斤）

调味：
- 蒜 4瓣（约15g），切末
- 孜然粉 1汤匙（约5g）
- 孜然粒 1茶匙（约2g，增加颗粒感）
- 辣椒面 1茶匙（约3g，按口味调整）
- 花椒粉 1/2茶匙（约1g）
- 盐 2-3g
- 生抽 2茶匙（10ml）
- 0卡糖 1/2茶匙（约2g，提鲜上色）
- 熟白芝麻 10g
- 葱花 2-3根
- 食用油 2汤匙（30ml）

做法：
1. **煮土豆**：小土豆洗净不去皮，冷水下锅，大火煮开后转小火煮15-20分钟（筷子能轻松穿透即熟，⚠️ 不要煮过头，否则一压就碎成泥）
2. **压扁**：捞出过凉水降温，可去皮也可带皮（带皮更香脆）。用刀背或手掌轻轻压扁每个土豆，**保持大致完整但裂开**（接触面大，煎时更容易焦脆）
3. **煎至焦脆**：平底锅倒2汤匙油，中火烧热，压扁的土豆平铺入锅（不要叠放），**不要频繁翻动**，煎至底面金黄焦脆再翻面，两面煎到焦黄，约5-8分钟
4. **爆香蒜末**：土豆推到一边，在锅中空位下蒜末爆香约15秒
5. **调味**：加盐、生抽翻炒均匀，撒入孜然粉、孜然粒、辣椒面、花椒粉、0卡糖，**大火快速翻炒**让调料均匀裹在土豆表面
6. **出锅**：关火，撒白芝麻和葱花翻匀装盘

关键技巧：
- **土豆别煮烂**：筷子刚好穿透就捞出，煮过头一压就碎
- **压扁力度**：用刀背轻压到裂开即可，别压成饼，保留厚度才外焦里糯
- **干煸不是油炸**：少油平底锅煎，但一定要耐心等焦壳形成再翻面
- **粉类调料后放**：孜然粉、辣椒面在土豆煎好后再放，否则容易糊锅发苦
- **不加太多液体调料**：生抽只需少量提鲜，加多了土豆吸水变软失去脆感
