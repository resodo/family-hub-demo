### 照烧鸡肉三明治

食材：去骨鸡腿2只、吐司、芝士片、蔬菜（牛油果/生菜/番茄）
照烧汁（2只鸡腿量）：生抽30ml、蚝油30ml、糖10g、日本料理酒30ml

1. 冷锅不放油，鸡腿带皮面朝下煎，煎到两面金黄
2. 下照烧汁，收到粘稠沾满鸡肉为止
3. 吐司两面放芝士片，中间夹蔬菜和切块鸡腿
4. 三明治机中火预热5分钟，放入三明治加热5分钟

### 隔夜燕麦（蓝莓柠檬版）

单人份

**基底（前晚混合）：**

| 食材 | 用量 | 说明 |
|------|------|------|
| 传统燕麦片 | 40g | 必须用传统燕麦片，即食燕麦会糊，钢切燕麦太硬 |
| 奇亚籽 | 12g（1汤匙） | 吸水膨胀形成凝胶质感，浓稠口感的关键 |
| 亚麻籽（磨碎） | 7g（1汤匙） | 必须用磨碎的亚麻籽，整粒无法消化吸收 |
| 希腊酸奶（无糖） | 60-70g | 蛋白质+奶香 |
| 蓝莓果酱 | 20g（1汤匙） | 提供甜味，可加至30g |
| 全脂牛奶 | 120ml | 喜欢稀一点可加到160ml |
| 盐 | 1小撮（约0.5g） | 提升风味，不可省 |

**出锅配料（第二天加）：**
- 新鲜蓝莓 35-40g，冷冻亦可
- 柠檬皮屑 少许（约半茶匙），现刮只取黄色部分

**做法：**
1. 密封罐中放燕麦+奇亚籽+亚麻籽粉，搅匀
2. 加酸奶+果酱+牛奶+盐，用叉子搅匀（确保果酱散开无干粉团块）
3. **静置5-10分钟后再搅一次**（防止奇亚籽结块沉底）
4. 盖紧冷藏过夜（至少4小时，最佳8小时）
5. 出冰箱观察浓稠度，太稠加一点牛奶拌匀。铺蓝莓，刮柠檬皮屑

口感技巧：
- 二次搅拌法是防结块关键
- 冷藏可保存3-5天，可以一次做3份备餐
- 不喜欢冷食可微波1-2分钟（配料加热后再放）
- 柠檬皮屑必须现刮，提前刮会氧化失香

**单份营养（约448千卡）：**

| 营养素 | 总量 | 占孕期每日推荐% | 孕期意义 |
|--------|------|----------------|----------|
| 蛋白质 | 约20g | 26% | 胎儿组织发育 |
| 膳食纤维 | 约12g | 42% | 预防孕期便秘 |
| 欧米伽3脂肪酸 | 约5.4g | >300% | 胎儿大脑视网膜发育（注：植物源转化率仅5-10%，不能替代鱼类） |
| 钙 | 约342mg | 34% | 胎儿骨骼发育 |
| 铁 | 约3.3mg | 12% | 预防孕期贫血 |

### 法式吐司

食材（每片）：厚吐司1片（**至少4cm厚**）、牛奶120ml、鸡蛋1个、糖10g、黄油适量

1. 牛奶、鸡蛋、糖充分混合打散
2. 厚吐司完全浸入蛋奶液（找一个和吐司差不多大小的容器，要完全浸没），冷藏浸泡8小时
3. 平底锅开小火，加黄油，放入吐司每面煎2分钟，上色到金黄
4. 空气炸锅180度，两面各加热6分钟

孕期超级食物搭配：配希腊酸奶 + 混合莓果（蓝莓/草莓/覆盆子）+ 奇亚籽 + 蜂蜜

关键：前一晚泡好冷藏，第二天早上直接煎+炸，10分钟搞定

### 香菇鸡肉粥

食材（2人份）：
- 大米 100g
- 鸡胸肉 100g（剁碎末）
- 鲜香菇 3-4朵（切2mm薄片）
- 生菜 3-4片（手撕小片）
- 姜 3-4片
- 葱花 适量

鸡肉腌料：盐 1g、玉米淀粉 5g、油 3ml、白胡椒粉少许，抓匀腌10分钟

1. 大米淘洗后清水浸泡30分钟。
2. 锅中烧开足量水（有高汤更好），下姜片+泡好的大米+1小勺油，大火煮开转小火煮30分钟，中途搅拌防糊底。
3. 粥煮至软烂后，下鸡肉碎迅速搅散，下香菇片，煮2-3分钟。
4. 关火，放入生菜碎，加盐调味，淋几滴香油，撒葱花。

关键技巧：
- 粥底先熬好再放料，配料煮制不超过3分钟，保持鲜嫩。
- 鸡肉提前腌淀粉，口感嫩滑不柴。
- 煮粥时加1小勺油，成品色泽鲜亮、入口更滑。
- 米提前泡30分钟，更容易煮至开花软烂。

### 鸡蛋菜饼（无面粉版）

食材（1-2人份）：
- 包菜 120g（切细丝，⚠️ 必须切丝不能切碎，碎了出水且口感差）
- 鸡蛋 3个
- 生抽 10ml（约2茶匙）
- 盐 1g（约1/4茶匙）
- 黑胡椒粉 少许
- 橄榄油 10ml

做法：
1. 包菜切细丝（约3mm宽），放入大碗
2. 打入3个鸡蛋，加生抽10ml、盐1g、黑胡椒，搅拌均匀让每根菜丝都裹上蛋液
3. 平底锅中小火热油10ml，倒入蛋菜糊，铺平
4. **小火煎，不要动**，等底面完全凝固定型（约3-4分钟）
5. 翻面再煎2-3分钟至两面金黄
6. 出锅切块

关键技巧：
- 蛋量要够：120g菜配3个蛋，蛋少了散不成饼
- 小火+不动是核心，大火或频繁翻动会散掉
- 也可以用西葫芦丝替换包菜丝，口感更嫩

### 孕期营养奶昔

模板：液体基底 200ml + 冷冻水果 150g + 蔬菜（可选）+ 蛋白质 + 可选加料

液体基底（选一）：
- 牛奶 200ml
- 希腊酸奶 100g + 牛奶 100ml
- 燕麦奶 200ml

4种组合：
**A. 香蕉苹果（百搭入门）**
- 冷冻香蕉 1根 + 苹果半个 + 菠菜一把（30g，味道被完全盖住）
- [叶酸/钾/维C]

**B. 莓果高纤**
- 冷冻混合莓果 150g + 奇亚籽 1汤匙
- [抗氧化/欧米伽3/纤维]

**C. 牛油果能量**
- 牛油果半个 + 冷冻香蕉 1根 + 可可粉 1茶匙
- [健康脂肪/叶酸/钾/镁]

**D. 苹果燕麦**
- 苹果半个 + 冷冻香蕉半根 + 即食燕麦 30g + 肉桂粉少许
- [纤维/铁/维C]

可选加料（每次选1-2种）：
- 亚麻籽粉 1汤匙（欧米伽3）
- 花生酱/杏仁酱 1汤匙（蛋白质+健康脂肪）
- 蜂蜜 1茶匙（调甜度）

做法：
1. 所有材料放入搅拌机
2. 高速搅打30-60秒至顺滑
3. 倒出即饮

注意：作为早餐饮品搭配主食，单喝蛋白质不够撑不到午饭

### 味增汤

食材（2人份）：
- 味增酱 30g（约1.5汤匙，推荐白味增/信州味增，味道柔和）
- 嫩豆腐 150g（切1.5cm小块）
- 干裙带菜 5g（泡发后约50g；超市搜"裙带菜"或"味增汤海带"）
- 水 500ml

做法：
1. 干裙带菜冷水泡5分钟至展开，沥干备用
2. 锅中加水500ml，大火煮开
3. 放入豆腐块，中火煮2分钟
4. 放入泡发的裙带菜，煮30秒
5. **关火**，等水温稍降（不再沸腾）
6. 取一小碗，舀2-3勺热汤，把味增酱放入小碗中用筷子搅散化开
7. 将化开的味增倒回锅中，轻轻搅匀

⚠️ 关键：味增酱绝对不能煮沸，否则变涩发苦，香气全失。必须关火后再加入

### 三文鱼茶泡饭（出汁版）

食材（1人份）：
- 米饭 1碗
- 三文鱼 100-120g
- 盐 少许 + 黑胡椒少许（腌鱼用）
- 海苔丝 适量
- 芝麻 少许
- 葱花 少许
- 芥末 少许（可选）

出汁 + 调味：
- 加强版出汁 200ml（做法见「统一出汁」）
- 淡口酱油 1茶匙
- 味淋 1茶匙

**统一出汁（一次做好，牛肉饭/茶泡饭/味增汤共用）：**
- 昆布 10g + 木鱼花 15g + 水 600ml
- 昆布冷水泡30分钟 → 中火加热至即将沸腾取出昆布（不要煮沸）→ 加木鱼花关火泡3-5分钟 → 过滤
- 出约500ml：牛肉饭取120ml，茶泡饭取200ml，剩余做味增汤或冷藏3天
- 木鱼花比标准多50%，浓度足够做茶泡饭汤底

做法：
1. **出汁调味**：取200ml加强版出汁，加淡口酱油和味淋调味
2. **煎三文鱼**：三文鱼抹盐和黑胡椒，平底锅中火煎至**两面全熟**（每面约3-4分钟），用筷子拨开确认无生肉。拨成块状
3. **组装**：碗中盛米饭，铺上三文鱼块，撒海苔丝、芝麻、葱花
4. **浇出汁**：热出汁沿碗边缓缓倒入，不要直接浇在鱼上（保持鱼表面口感）

⚠️ 孕期：三文鱼必须全熟，不能半生

### 自制牛肉饼（批量冷冻版）

食材（约8个饼）：
- 牛肉馅 500g（肥瘦比 2:8 或 3:7，太瘦会柴）
- 洋葱 50g（切极细末）
- 鸡蛋 1个
- 面包糠 20g
- 牛奶 20ml（泡软面包糠用）
- 盐 3g
- 黑胡椒粉 适量
- 蒜 2瓣（切末）

做法：
1. 面包糠加牛奶20ml拌匀，静置5分钟让面包糠吸饱奶（锁住水分的关键）
2. 牛肉馅加盐3g、黑胡椒、蒜末，**顺时针搅拌**至起粘性（约2-3分钟）
3. 加入鸡蛋、泡好的面包糠、洋葱末，继续顺时针搅拌均匀
4. 取一团肉（约60-70g），左右手来回**摔打5-6次**排出空气，整成圆饼状（约1cm厚，中间略按薄防鼓包）
5. 每个饼之间隔油纸/保鲜膜，叠放入密封袋，**冷冻保存**

煎法：
1. 冷冻饼无需解冻，平底锅中火热少许油
2. 放入肉饼，中火煎3-4分钟至底面焦褐
3. 翻面，转中小火再煎4-5分钟至全熟（中心无粉红）
4. ⚠️ 孕期必须全熟，可用温度计确认中心71°C+

关键技巧：
- 顺时针搅拌上劲 = 口感紧实弹牙，乱搅 = 松散
- 摔打排气 = 煎的时候不裂不鼓
- 面包糠+牛奶 = 保湿，没有这步肉饼会干柴
- 一次做8个冷冻，早餐随取随煎很方便

### 大早餐

框架式早餐，蛋白质三选一 + 固定配菜

蛋白质（选一）：
- 自制牛肉饼（冷冻取出直接煎）
- 香煎鱼排（龙利鱼/三文鱼/鳕鱼）
- 农协煎鸡（需提前48小时准备）

固定配菜：
- 煎蛋 1个
- 烤番茄：小番茄对半切，空气炸锅180°C 8-10分钟
- 煎蘑菇：口蘑切片，少许油中火煎至金黄（约3-4分钟）
- 牛油果：半个，切片
- 吐司 1-2片
- 生菜几片

做法：
1. 空气炸锅先放番茄（180°C 8-10分钟，最耗时先启动）
2. 同时平底锅煎蛋白质主菜
3. 主菜煎好盛出，原锅煎蘑菇+煎蛋
4. 吐司烤一下，牛油果切片，生菜铺盘
5. 全部摆盘上桌

注意：
- 不用培根/香肠/茄汁焗豆
- 周末在家吃的坐下来享受型早餐，不适合带走
- 蛋白质提前确认：牛肉饼随时可煎，鱼排也随时可煎，农协鸡需48小时准备

### 费城牛肉芝士三明治（热压版）

食材（2-3份）：
- 牛肉薄片 300g（肥牛/肋眼薄切，或直接买烤肉片）
- 洋葱 半个（切丝）
- 彩椒 1个（切条，或青椒也行）
- 蒜 2瓣（切碎）
- 原制芝士片 3-4片（普罗沃洛奶酪最正宗，切达/马苏也行）
- 法棍面包 1根（或厚吐司对切）
- 黄油 10g（涂面包用）
- 橄榄油 适量
- 盐、黑胡椒 适量

做法：
1. 热锅加橄榄油，下洋葱丝炒至半透明出香，加彩椒条+蒜碎+盐+黑胡椒炒匀，盛出备用
2. 锅中补少许油，牛肉片铺平不翻动，撒盐和黑胡椒，煎至上色翻面（不要反复翻，焦香感很重要）
3. 牛肉煎好后把蔬菜倒回锅中混合，**熄火**，铺上芝士片，盖锅盖利用余温融化芝士（约1分钟）
4. 法棍对半剖开不切断，抹黄油。两种加热方式：
   - **三明治机**：预热5分钟，法棍放入压3-4分钟至金黄酥脆
   - **空气炸锅**：180°烤3-4分钟至表面金黄酥脆
5. 牛肉蔬菜芝士夹入面包，合起即可

关键技巧：
- 牛肉要薄切才好吃，冷冻30分钟后更容易切薄；懒人直接买肥牛卷/烤肉片
- 芝士用余温融化，不要大火烧，否则出油变硬
- 法棍要烤到外酥内软，喷点水防止太干
- 可提前做好冷藏，第二天空气炸锅180°复热10分钟或三明治机复热5分钟

### 奶油菠菜三文鱼三明治

⚠️ 孕期：三文鱼必须全熟，不能用烟熏三文鱼（冷熏未煮熟有李斯特菌风险）

食材（2人份）：

三文鱼：
- 三文鱼排 2块（每块约150g），去皮
- 橄榄油 1汤匙（15ml）
- 盐 1/2茶匙、黑胡椒适量
- 柠檬汁 1茶匙（煎好后挤）

奶油菠菜：
- 菠菜 300g
- 黄油 15g（1汤匙）
- 蒜 2瓣切末
- 奶油奶酪 60g
- 牛奶 30ml（2汤匙，帮助化开奶酪调稀；有淡奶油更好）
- 盐、黑胡椒适量
- 肉豆蔻粉 一小撮（可选，有的话味道更正宗）

组装：
- 法棍面包 1根（或吐司 4片）
- 番茄 1个（切片）
- 黄油 10g（煎面包用）

做法：
1. **奶油菠菜**：菠菜洗净，烧开水焯1分钟捞出，过凉水，使劲挤干水分（⚠️ 关键：水分不挤干最后会稀），切碎。锅里放黄油15g小火融化起泡，下蒜末小火炒30秒出香（不能焦）。加入奶油奶酪和牛奶，小火不停搅拌至奶酪完全融化成顺滑的酱。加入切碎的菠菜翻拌均匀，小火煮2-3分钟至浓稠。加盐、黑胡椒、肉豆蔻粉调味，盛出备用
2. **煎三文鱼**：两面撒盐+黑胡椒，中火加橄榄油，每面煎3-4分钟至全熟（中心≥63°C），出锅挤柠檬汁，用叉子拆成大块
3. **组装**：法棍面包斜切成两段，每段从侧面剖开。两面都厚厚涂一层奶油菠菜，铺番茄片，放三文鱼块，合起来
4. **煎**：平底锅中小火放黄油10g，法棍面包切面朝下放入，压着煎2-3分钟至切面金黄酥脆，翻面再煎1-2分钟。吐司版可用三明治机热压3-4分钟

关键技巧：
- 菠菜必须挤干水分，否则奶油菠菜太稀涂不住
- 奶酪小火慢慢化，大火会结块
- 奶油菠菜可以前一晚做好冷藏，第二天直接用
- 便当版：不煎，奶油菠菜单独小盒带，到公司组装

### 炝锅面（特厨隋坡版）

食材（2人份）：
- 大葱 2-3根（要多！主要用葱白，斜刀切成眉毛粗细）
- 鸡蛋 4个
- 西红柿 1个（切块）
- 面条 半斤（手擀面/鲜切面最佳，不要用干面条）
- 酱油（生抽） 约10g（2勺）
- 八角 1颗（可不放）
- 盐 适量（出锅前加）
- 食用油 要多

做法：
1. **炒鸡蛋**：锅内多加油，大火烧至八成热，倒入蛋液，蛋液迅速膨胀，**一定要炒老一些**，炒出焦香味，盛出备用
2. **煸葱炝锅（灵魂步骤）**：锅内再加油，放入斜切大葱（+1颗八角可选），**中小火慢煸**至大葱微微焦黄发深色（不能真糊，金黄到微深色是最佳状态）
3. **炒番茄**：加入西红柿块大火翻炒，烹入酱油约2勺，翻炒出焦香味
4. **加水煮汤**：趁热加入**热水**（不能冷水），倒回炒好的鸡蛋，**大火滚煮3-4分钟**让蛋香、葱香、番茄酸甜充分融合
5. **煮面**：下入面条用筷子搅散，开锅后煮约5分钟（根据面条粗细调整），出锅前加盐调味

两个核心窍门：
- **鸡蛋要炒"老"**：油多、油温高，炒出焦香味才是汤底鲜美的来源
- **大葱要"多"**：大葱斜刀切，中小火慢煸到金黄，葱油香气是这碗面的魂。必须用大葱不能用小葱

其他技巧：
- 加水一定用热水，冷水降温影响锅气
- 煮面会吸汤，水要留够，喜欢稀的多放水
- 不需要加味精/鸡精/胡椒粉，盐和酱油就够

### 生滚鱼片粥

食材（2人份）：
- 大米 100g
- 鲈鱼/龙利鱼片 150g（斜刀切薄片，约3mm）
- 姜丝 10g
- 葱花 适量
- 香菜（可选）

鱼片腌料：盐 1g、白胡椒粉少许、玉米淀粉 5g、油 3ml，轻抓匀腌10分钟

做法：
1. 大米淘洗，加水浸泡30分钟，沥干后拌入1小勺油，冷冻6小时以上（冻米煮粥5分钟即绵稠，可提前冻好备用）
2. 锅中烧开水约1200ml，下冻米+姜丝，大火煮开转中小火，不断搅拌煮15-20分钟至米粒开花绵稠
3. 粥底煮好后大火，下鱼片逐片铺开，不要搅动，煮约30秒至鱼片变白卷起即熟
4. 关火，加盐调味，撒葱花、香菜，淋几滴香油

关键技巧：
- 冻米是广式快速煮绵粥的秘诀，米粒冻裂后煮制更快开花
- 鱼片要切薄+腌淀粉，入粥后嫩滑不散
- 鱼片下锅后不要乱搅，轻推即可，否则碎掉
- 鱼用鲈鱼最鲜（刺少），龙利鱼也行但鲜味略弱
- ⚠️ 孕期注意：鱼片必须完全煮熟变白，不能半生

### 日式牛肉饭

**2人份 | 约15分钟（不含煮饭）**

食材：
| 食材 | 用量 | 备注 |
|------|------|------|
| 牛肉薄切片（肥牛卷） | 340g | 2mm厚度，火锅肥牛卷直接用 |
| 洋葱 | 1/2个（约150g） | 顺纹切1cm宽月牙片 |
| 小葱 | 1根 | 切葱花，最后撒 |
| 米饭 | 2碗 | |

酱汁：
| 调料 | 用量 |
|------|------|
| 加强版出汁 | 120ml（做法见「统一出汁」） |
| 酱油 | 3汤匙（45ml） |
| 味淋 | 2汤匙（30ml） |
| 清酒 | 2汤匙（30ml） |
| 0卡糖 | 1汤匙量 |

做法：
1. **冷锅调酱汁**：所有酱汁材料倒入平底锅（不开火），搅匀使糖溶解
2. **煮洋葱**：洋葱片铺入酱汁，**盖盖子**中火煮沸，转中小火煮3-4分钟至半透明变软
3. **煮牛肉**：肥牛片一片片摊开铺在洋葱上（不要堆叠），盖盖子中小火煮。1-2分钟后开盖拨散，**撇浮沫**。肉变色即熟，**不要久煮**——变色后再煮超过1-2分钟就变硬变柴
4. **装碗**：碗里盛热米饭，先浇一点酱汁让饭吸汁，铺牛肉和洋葱，再浇酱汁，撒葱花

关键技巧：
- **肉要薄（2mm）**：厚肉片短时间煮不嫩
- **冷锅起步**：酱汁冷锅混合均匀，避免局部焦糖化
- **洋葱先煮、肉后放**：洋葱需3-4分钟，肉只需2-3分钟
- **肉变色立刻停**：过度煮制是牛丼变难吃的第一杀手
- **盖盖子煮**：防止蒸发变咸

### 番茄腌肉意面

**份量：2人份 | 总时间：约30分钟**

食材：

主料：
| 食材 | 用量 | 备注 |
|------|------|------|
| 空心面 或 通心管面 | 200g | 传统用空心面，通心管面也正宗 |
| 猪脸颊肉（风干猪颧肉） | 100g | 买不到可用意式风干五花肉代替，不要用培根（培根是烟熏的，风味完全不同） |
| 罐装去皮番茄（推荐意大利圣马扎诺品种） | 400g（1罐） | 用一半约200g，或整罐做浓一点 |
| 罗马羊奶硬奶酪 | 50g | 现刨细丝/细粉，硬奶酪孕期安全 |

调味料：
| 食材 | 用量 | 备注 |
|------|------|------|
| 干辣椒碎 | 1/2 茶匙（约1g） | 或1根干辣椒，可按口味增减 |
| 干白葡萄酒 | 30ml | 可选，会完全挥发酒精 |
| 盐 | 适量 | 羊奶酪本身很咸，酱汁少放盐 |
| 黑胡椒 | 适量 | 现磨 |

**正宗版不放的东西：** 洋葱、大蒜、橄榄油（猪脸颊肉自己出油足够）、糖（好的罐装番茄不需要；偏酸可加极少量零卡甜味剂）

做法：

1. **处理猪脸颊肉**：去掉外皮（黑色胡椒壳层可保留），先切成约5mm厚片，再切成5mm × 3cm的小条。保留肥瘦相间部分，肥肉是出油和风味的核心
2. **冷锅中小火煎猪脸颊肉（灵魂步骤，8-10分钟）**：冷锅放入肉条，**中小火**起煎，不加任何油。偶尔翻动，煎至**边缘金黄微脆、肥肉半透明**。中途加入辣椒碎一起炒出香味。**不能大火，否则外焦内生、出油不充分**
3. **加酒（可选）**：倒入30ml干白葡萄酒，让酒精完全挥发（约1-2分钟）
4. **煮番茄酱（10-15分钟）**：罐装番茄倒入碗中，**用叉子压碎**成粗块状（不要打成泥，保留质感）。连汁倒入锅中翻拌均匀，**中小火慢炖10-15分钟**至略微收浓、油脂和番茄融合。**少量加盐**
5. **煮面（与第4步同时进行）**：大锅烧5L水加盐（每升水约10g盐），水大开后下面，**比包装时间少煮1分钟**。**留出200ml煮面水**再沥干
6. **合体（2-3分钟）**：面条倒入番茄酱锅中，小火，加入2-3大勺煮面水，快速翻拌30-60秒裹酱。**关火**，撒入约2/3的羊奶酪碎，快速翻拌均匀 → 淀粉水 + 奶酪 + 油脂形成**乳化的浓稠酱汁**。太干再加一点煮面水
7. **装盘**：用夹子卷面装盘，浇锅底剩余酱汁，顶部撒剩余羊奶酪碎和现磨黑胡椒，**立即食用**

关键技巧：
- **猪脸颊肉冷锅中小火慢煎**是整道菜的核心，急不得
- **番茄用叉子压不用搅拌机**，保留粗糙质感才正宗
- **煮面水是天然乳化剂**，淀粉水让酱汁变得丝滑浓稠
- **羊奶酪关火后再加**，避免高温结块
- **不加洋葱、不加蒜、不加橄榄油** — 正宗版本的三不原则

⚠️ 孕期安全：猪脸颊肉经充分煎煮全熟安全；罗马羊奶硬奶酪是硬质陈年奶酪（至少8个月），孕期可放心食用；白葡萄酒加热后酒精完全挥发

### 泰式打抛猪肉饭

**2人份 | 约15分钟**

主料：
- 猪肉末（或鸡肉末）600g
- 九层塔（罗勒叶）1大把约30g，只摘叶子
- 豇豆（长豆角）100g，切小丁（约0.5cm）
- 大蒜 5-6瓣，拍碎粗切（不要切太细，粗蒜粒大火下产生焦香是泰式精髓）
- 小米辣 2-4根切碎（怕辣可去籽）
- 鸡蛋 2个
- 米饭 2碗

调味料（提前混合）：
| 调料 | 用量 | 备注 |
|------|------|------|
| 生抽 | 1.5汤匙（22ml） | |
| 鱼露 | 1汤匙（15ml） | 鱼露是打抛灵魂 |
| 蚝油 | 1汤匙（15ml） | |
| 老抽 | 半茶匙（2-3ml） | 只调色，多了会黑 |
| 0卡糖 | 1茶匙 | |
| 清水 | 2汤匙（30ml） | |

用油：共约3汤匙（煎蛋1汤匙+炒肉2汤匙）

做法：
1. **煎蛋**：中火热锅加1汤匙油，鸡蛋煎至**两面全熟**（孕期不做溏心），边缘可微焦，盛出
2. **爆香**：同锅大火加2汤匙油至微冒烟，下蒜碎+小米辣大火爆香30秒至蒜微金黄
3. **炒肉**：下猪肉末**大火翻炒**，锅铲打散，炒至完全变色无粉红（3-4分钟）。**出水就继续大火炒干，要听到滋滋声**。油太多适当倒掉一些
4. **加豇豆**：下豇豆丁翻炒1-2分钟
5. **调味收汁**：倒入混合调味料，大火翻炒让酱汁收浓裹住肉末（1-2分钟）
6. **关火放九层塔**：关火后放入九层塔叶，余温快速翻拌均匀（九层塔久炒会发黑变苦）
7. **装盘**：米饭倒扣盘中，打抛肉浇旁边，煎蛋放最上面

关键技巧：
- **全程大火**，肉末要炒到焦香有锅气，不能水煮般软塌
- **蒜拍碎粗切不要蒜泥**，粗蒜粒大火下焦香是泰式精髓
- **九层塔最后关火放**，高温会迅速变黑变苦
- **老抽只放半茶匙**，正宗只是几滴调色，多了颜色发黑
- 正宗用圣罗勒（打抛叶），国内九层塔可替代

⚠️ 孕期：鸡蛋必须全熟；九层塔孕期安全可食用；用0卡糖代替白糖

### 蒜香黄油蛤蜊意面

**2人份 | 约25分钟（不含吐沙）**

主料：
- 蛤蜊（花蛤/文蛤均可）500g
- 意面（细面/直面）200g

调味/配料：
| 食材 | 用量 | 备注 |
|------|------|------|
| 大蒜 | 4-5瓣（约20g） | 切薄片，不要切碎容易糊 |
| 无盐黄油 | 15g | 增香增亮 |
| 橄榄油 | 3汤匙（45ml） | 开蛤蜊1汤匙+炒蒜2汤匙 |
| 干白葡萄酒 | 80ml（约5汤匙） | 开蛤蜊用，酒精高温挥发 |
| 新鲜欧芹 | 15g（切碎） | 没有可省 |
| 干辣椒碎 | 半个量 | 可不放 |
| 柠檬 | 半个 | 取皮屑+汁，提亮香气 |
| 盐、现磨黑胡椒 | 适量 | 蛤蜊本身有咸味，盐要酌情 |

**不放的东西：** 奶油、芝士、味精/鸡精、蚝油（海鲜意面加奶油/芝士是大忌，会压制鲜味）

**前期准备：蛤蜊吐沙（提前1-2小时）**
1. 逐个检查，轻敲后开口不合拢的扔掉（死蛤蜊）
2. 盆中清水加2汤匙盐模拟海水，蛤蜊放入，**水不要完全没过**，留一半露出水面（蛤蜊会更活跃吐沙）
3. 静置1-2小时，换水1-2次
4. 反复冲洗2-3次至水中完全无沙

做法：
1. **煮面**：大锅水开加一大勺盐（水要有明显咸味），下意面**比包装时间少煮2分钟**。捞面前**留出1碗煮面水**备用
2. **开蛤蜊取汁**：另起大平底锅中大火，倒1汤匙橄榄油，放入蛤蜊+白葡萄酒80ml，盖盖，每20秒晃一下锅，约2-3分钟蛤蜊全部开口。**蛤蜊肉从壳里剥出，壳扔掉**。汤汁用细网**过滤一次去残沙**，肉和汤分开放。始终不开口的直接扔掉
3. **炒蒜香底料**：同锅擦干，**小火**倒2汤匙橄榄油+黄油15g，让黄油慢慢融化起泡。放入蒜片小火慢煸至金黄飘香（约1-2分钟），放辣椒碎（可选）。**蒜焦黑=发苦，全程小火耐心等**
4. **合体收汁（灵魂步骤）**：倒入过滤好的蛤蜊汤汁，中火煮开。放入意面，用夹子不停翻拌，分次加煮面水（每次2-3汤匙）边加边快速翻拌，约2分钟让面充分吸汁、酱汁乳化浓稠。挤入柠檬汁（约1汤匙）+刨少许柠檬皮屑，尝咸淡调盐
5. **出锅**：蛤蜊肉倒回锅中轻拌均匀，撒欧芹碎+黑胡椒，淋少许橄榄油，装盘

关键技巧：
- **蛤蜊分两次进锅**：先开壳取汁剥肉，最后再放回，避免反复加热肉变老
- **煮面水是乳化关键**，淀粉水让酱汁浓稠挂面
- **蛤蜊汤汁必须过滤**，去残沙保口感干净
- **蒜片小火慢煸**，金黄即可绝不能焦
- **柠檬皮屑**可选但推荐，能提亮整道菜的香气层次

⚠️ 孕期安全：蛤蜊属低汞贝类，孕期可安全食用；必须全部开口全熟，不开口的扔掉；白葡萄酒高温后酒精完全挥发

### 那不勒斯海鲜意面（阿达原版，2人份）

主料：
- 意面 200-250g（空心面或细面均可）
- 蛤蜊 300g（提前盐水吐沙2小时）
- 青口贝 300g（刷干净外壳拔掉足丝）
- 虾 200g（去壳去虾线）
- 鱿鱼 150g（切圈或切条）

配料：
- 小番茄 200g（对半切）
- 番茄膏 1汤匙（15g，可选，想要番茄味更浓郁时加）
- 蒜 4瓣（切末）
- 干辣椒 1-2根（可选，掰碎去籽）
- 白葡萄酒 150ml
- 橄榄油 30ml
- 新鲜欧芹 1把（切碎，分两半）
- 新鲜罗勒 5-6片（手撕）
- 盐 适量（轻手，贝类本身咸）
- 黑胡椒 多放

做法：
1. **爆香**：橄榄油30ml下锅，中火加入蒜末和干辣椒碎爆香至蒜微金黄
2. **先下贝类**：加入蛤蜊和青口贝，盖盖煮至贝壳开口。**没有开口的全部挑出扔掉**（已经死了不能吃）。开口后贝肉和汤汁分开：贝肉取出备用，汤汁留在锅里
3. **下其他海鲜**：虾和鱿鱼下锅翻炒至变色（虾变红、鱿鱼变白，约2分钟）
4. **加酒加香草**：加入一半的欧芹碎和罗勒，倒入白葡萄酒150ml，中火煮至酒精蒸发（约2-3分钟，闻不到酒味为止）
5. **加番茄调味**：加入对半切的小番茄（想更浓郁可加1汤匙番茄膏），加盐（轻手）和黑胡椒（多放）调味。把贝肉倒回锅里
6. **煮意面**：大锅水烧开加盐加少许橄榄油，意面按包装时间**少煮1分钟**（比如标9分钟就煮8分钟）
7. **合体**：意面捞入海鲜锅，大火翻拌1分钟让面吸汁，关火撒剩下一半欧芹碎，出锅

关键技巧：
- 贝类开口后壳扔掉、肉和汤分开，最后再合体，避免贝肉煮老
- 意面少煮1分钟在海鲜锅里完成最后熟化，口感更好
- 盐一定要轻手，蛤蜊青口本身带大量咸鲜味
- 白葡萄酒酒精烹饪会完全蒸发，孕期安全

⚠️ 孕期安全：蛤蜊、青口、虾、鱿鱼均为低汞海鲜，孕期可安全食用；所有海鲜必须全熟；不开口的贝类扔掉

### 咸蛋黄海鲜粥

⚠️ 孕期暂不安排（避免腌制类食品），产后再做

食材（2-3人份）：
- 大米 150g（提前浸泡1小时+）
- 鲜虾 300g（活虾最佳，有虾膏更好）
- 咸鸭蛋 2个（熟的，蛋黄碾碎，蛋白切小块，分开备用）
- 香菇 3-4朵（切薄片）
- 油菜/上海青 2棵（切段）
- 姜 3片
- 料酒 15ml（1汤匙）
- 盐 适量（咸蛋黄有盐，少放）
- 白胡椒粉 2g（1/2茶匙）
- 食用油 15ml（1汤匙）
- 水 900-1050ml（米水比1:6-1:7）

做法：
1. **处理虾**：去虾线，掰下虾头，剥壳留虾仁+虾壳。虾头内黑色沙囊必须去掉，保留橘红色虾膏。虾仁用料酒5ml+白胡椒粉1g腌10分钟
2. **炒虾油+咸蛋黄（灵魂步骤）**：小火加油15ml放姜片煸香，放虾头虾壳**小火慢慢压炒**3-4分钟至虾头变红、锅底出橘红色虾油。加碾碎的咸蛋黄小火翻炒至起沙冒泡（1-2分钟）。**必须先炒虾头出虾油再加咸蛋黄，顺序反了效果差很多**
3. **熬粥底**：加900-1050ml**开水**（必须开水），大火煮沸，捞出虾头虾壳。倒入泡好的大米，大火煮沸转小火熬15-20分钟至米粒开花绵稠，偶尔搅拌防粘底
4. **加配料（注意顺序）**：先放香菇片煮3-4分钟 → 再放虾仁煮2分钟 → 最后放油菜段和咸蛋白碎煮1分钟
5. 加盐调味（先尝再加，咸蛋黄已有咸味），撒白胡椒粉，关火淋几滴芝麻油

关键技巧：
- **虾头小火压炒出虾油**是"鲜美"vs"普通"的关键，大火会焦不出油
- 虾仁最后放，煮2分钟足够，久了老
- 去腥三件套：姜片+料酒腌虾仁+虾头去沙囊
- 可加3-5颗泡发干贝一起熬，更鲜

### 潮汕海鲜砂锅粥（虾蟹版）

食材（2-3人份）：

主料：
- 大米 110-120g（珍珠米最佳）
- 活虾 200g（剪须去虾枪，虾头剪下备用）
- 螃蟹 1只（约200g，花蟹/梭子蟹）
- 淡干干贝 4粒（8-10g，温水泡20分钟撕丝）
- 姜 1块（约7g，切细丝）
- 猪油 1勺（15g，没有可用花生油代替）
- 水 1700ml
- 花生油 15ml（爆虾头用）

调味：
- 冬菜 15g（切碎）— 买不到可用少许榨菜碎
- 花生酱（纯/柔滑型）半勺（10g）
- 盐 约5g（分次加，出锅前调整）
- 白胡椒粉 1/3茶匙（0.5g）
- 芝麻香油 几滴（3g）
- 香菜末 10g（或芹菜丁/葱花）

做法：
1. **炒米（起胶关键）**：空锅不放油，大火翻炒大米至个别米粒微微发黄，盛出备用
2. **爆虾头出虾油**：砂锅热锅下花生油15ml+姜丝，放虾头**小火煎炒至金黄出橘红色虾油**（3-4分钟），用铲子按压虾头挤出虾膏
3. **处理螃蟹**：掀开蟹壳冲洗，去胃、蟹嘴、蟹腮，剪成四块，蟹钳刀背敲碎，保留蟹盖
4. **熬粥底**：往虾油锅中加1700ml**开水**，煮3-4分钟后捞出虾头。倒入炒好的大米+干贝丝+猪油15g，大火搅拌煮沸，转小火（让粥翻腾但不溢出），盖锅盖留缝，熬**20分钟**
5. **加海鲜和调味**：保持小火，放入螃蟹+花生酱10g+冬菜15g+盐5g+白胡椒粉，搅匀后转中大火边煮边搅**2分钟**
6. **下虾收尾**：螃蟹熟后放入虾，煮至虾变红（约2-3分钟），滴芝麻香油，撒香菜末，搅匀出锅

关键技巧：
- **炒米至微黄**是潮汕经典起胶做法，米粒外层淀粉破碎更快出胶
- **虾头爆虾油**是鲜味底层，必须煎到金黄冒泡
- **花生酱+冬菜**是潮汕砂锅粥的共识灵魂调料
- 干贝+猪油补天然鲜味，不用味精/鸡精
- 虾煮至变红即停，太久肉质变老
- 冬菜和干贝本身有咸味，盐量最后调整

### 韩式鲍鱼粥（济州版绿色粥底）

⚠️ 鲍鱼属低汞贝类，煮熟后孕期可食用（美国食药监建议每周≤1份）

食材（2人份）：
| 食材 | 用量 | 备注 |
|------|------|------|
| 粳米/短粒米 | 90g（1/2杯） | 提前泡至少1小时 |
| 鲜鲍鱼 | 2只（每只约100-110g带壳） | 必须新鲜活鲍 |
| 芝麻油 | 1汤匙（15ml） | 炒米和内脏用 |
| 水 | 720ml（3杯） | |
| 盐 或 汤酱油（국간장） | 约1/2-1茶匙 | 最后调味 |
| 小葱 | 1根（切葱花） | 出锅撒 |
| 熟白芝麻 | 少许 | 装饰 |

做法：
1. **处理鲍鱼**：刷洗外壳，用勺子撬下鲍鱼肉。**分离绿色内脏**（切下剁碎成糊状）——内脏是正宗济州版绿色粥底的来源。切掉口器，鲍鱼肉切5mm薄片
2. **芝麻油炒米+内脏（灵魂步骤）**：厚底锅放沥干的米+剁碎内脏搅匀，加芝麻油15ml，**中火翻炒约5分钟**至米粒半透明。加鲍鱼肉片继续炒2分钟
3. **煮粥**：加水720ml中大火煮开，转中火保持微沸，煮**约25分钟**，**后期勤搅拌**防粘糊底。太稠可加半杯水（120ml）
4. **调味**：关火前加盐或汤酱油（从少量开始约1/2茶匙，尝味再加）。盛碗撒葱花和白芝麻

关键技巧：
- **内脏一定要加**：不加=白色普通粥，加了才是正宗绿色济州鲍鱼粥
- **芝麻油炒米**是韩式粥的标志步骤，带出坚果香
- 米必须提前泡1小时+，粥才绵滑
- **最后才调味**，提前加盐粥会出水
- 鲍鱼不要切太碎，5mm片保留口感

- [Maangchi - Jeonbokjuk](https://www.maangchi.com/recipe/jeonbokjuk)

### 生滚牛肉粥

食材（2人份）：
- 大米 100g
- 牛肉 150g（肥牛片或牛里脊逆纹切薄片）
- 姜丝 10g
- 葱花 适量

牛肉腌料：生抽 10ml、玉米淀粉 5g、油 5ml、白胡椒粉少许、沙茶酱 5g（可选，增加潮汕风味），抓匀腌15分钟

做法：
1. 大米淘洗，加水+1小勺油，冷冻6小时以上（同鱼片粥的冻米法）
2. 锅中烧开水约1200ml，下冻米+姜丝，大火煮开转中小火搅拌煮15-20分钟至绵稠
3. 粥底煮好后转大火，下牛肉片逐片铺入，轻推不要猛搅，煮约30-40秒至变色即熟
4. 关火，加盐调味，撒葱花，淋几滴香油

关键技巧：
- 牛肉切薄+腌淀粉是嫩滑关键，下锅后不超过40秒，过久就老
- 用肥牛片最省事（不用自己切），里脊片口感更嫩
- 沙茶酱是潮汕做法的灵魂，不加也好吃但少一层风味
- 冻米技巧同鱼片粥，可以一次冻几份备用
