# 海鲜菜谱

### 那不勒斯炸海鲜

食材：斑节虾、沙虾、鱿鱼
裹粉：面粉20g、玉米面30g、面包糠30g、柠檬皮（擦丝器擦）
蘸酱：蛋黄酱+柠檬汁+红椒粉+蒜蓉

1. 虾去壳去虾线开背，留尾巴。虾和鱿鱼一起用白葡萄酒、海盐、黑胡椒腌底味（盐不要太多，后面蘸酱调咸度）
2. 裹粉混合均匀。海鲜擦干水分，蘸裹粉，用力按压让粉牢固附着
3. 空气炸锅底部喷油（花生油或牛油果油，190°C高温不宜用橄榄油），海鲜平铺一层不堆叠（一锅炸不下就分多锅），放好后再喷一层油
4. 空气炸锅 190度 10分钟
5. 出锅撒少量欧芹碎
6. 蘸酱调好，蘸着吃

注意：等小鱼到家再炸，提前炸了不好吃

### 巴西椰浆炖海鲜

两人份
食材：虾+鳕鱼共约1kg、蒜、洋葱、番茄、彩椒、高汤1杯、椰浆350g、红棕榈油、香菜、青柠檬

1. 虾开背去虾线，鳕鱼切3cm大方块，用青柠檬汁+盐腌制
2. 蒜切末，洋葱切丁，番茄切大块，彩椒切条
3. 中小火下一大勺红棕榈油，化开后炒蒜末→洋葱炒软→下彩椒和番茄→加高汤和椰浆，盐和黑胡椒调味
4. 下虾煮3分钟，再下鳕鱼煮3分钟
5. 出锅前加一勺红棕榈油和香菜段

### 蒜蓉粉丝蒸虾

食材（两人份）：大虾1斤、粉丝50g（1小把）、蒜1整头（约10瓣）、小葱2根
调味：生抽30ml（2汤匙）、蚝油15ml（1汤匙）、白糖5g（1茶匙）、食用油30ml（2汤匙，炒蒜用）

1. 粉丝提前温水泡软（约15分钟），沥干铺在盘底
2. 虾剪须脚，开背去虾线，铺在粉丝上
3. 蒜去皮全部切末（整头都用上，蒜蓉蒸虾的灵魂就是蒜要多）
4. 冷锅下油30ml，小火炒蒜末，炒到微微金黄出蒜香（火不能大，蒜容易糊）
5. 关火，蒜末里加生抽30ml+蚝油15ml+糖5g，拌匀成蒜蓉汁
6. 蒜蓉汁均匀浇在虾和粉丝上
7. 锅里水烧开，放入蒸盘，**大火蒸6-7分钟**（虾大的话7分钟，不要超时会老）
8. 出锅撒葱花

注意：必须等人快到家再蒸，蒸虾放了不好吃。粉丝会吸汁，这是精华

### 葱姜炒芒果贝

食材（2人份）：
- 芒果贝 500g
- 姜 3-4片（约15g）
- 小葱 3-4根（葱白葱绿分开切段）
- 橄榄油 2汤匙（约25ml）

调味：
- 料酒 1.5汤匙（约20ml）
- 生抽 1汤匙（约15ml）
- 白糖 1/2茶匙（约2g，提鲜）
- 盐 一小撮（约1g，贝类本身有咸鲜味，少放）

1. 芒果贝提前盐水泡2-3小时吐沙（水里加少许盐和几滴香油），洗净沥干
2. 大火热锅下油，五成热下姜片+葱白段爆香
3. **大火**下芒果贝快速翻炒，立刻烹入料酒
4. 加生抽、盐、糖翻匀
5. 炒至芒果贝**全部开口**即熟（约1.5-2分钟），撒葱绿翻两下马上出锅

关键技巧：
- 全程大火，开口即熟，多炒一秒肉就老
- 芒果贝本身比较干净，不需要泡太久，不放心就泡一下
- 不需要加水，贝类自身会出汁
- 姜是去腥关键，不能省

参考来源：[下厨房](https://m.xiachufang.com/recipe/104486017/)、[豆果美食](https://m-pr.douguo.com/cookbook/773927.html)、[食谱大全](https://shipu.bmcx.com/jiangbaomangguobei__shipuchaxun/)

### 味噌烤三文鱼

⚠️ 孕期：三文鱼必须烤至内部温度 **145°F / 63°C**（全熟），建议用食品温度计确认

食材（2人份）：
- 带皮三文鱼排 2块（每块约170g，厚度约2.5cm）

味噌腌料：
| 食材 | 用量 | 备注 |
|------|------|------|
| 白味噌 | 3汤匙（54g） | 白味噌最温和不过咸 |
| 味淋 | 1.5汤匙（22ml） | 提供甜味和光泽 |
| 清酒或料酒 | 1.5汤匙（22ml） | 去腥增香 |
| 砂糖 | 1汤匙（12g） | 帮助焦糖化上色 |
| 低钠酱油 | 1汤匙（15ml） | 增加咸鲜层次 |
| 芝麻油（熟榨） | 1/4茶匙（1ml） | 提香，不可多 |

**1. 腌制（30分钟-3小时）**
1. 三文鱼用厨房纸吸干表面水分
2. 所有腌料搅拌至味噌和糖完全溶解
3. 留出约1汤匙腌料备用（烤完刷面用）
4. 剩余腌料均匀涂抹在鱼肉面（非皮面），保鲜膜包好冷藏
5. **不要超过3小时**，味噌高盐分会过度脱水

**2a. 烤箱烘烤**：200°C / 400°F
1. 烤箱预热200°C，烤盘铺锡纸刷薄油
2. **关键：用手/纸巾将鱼表面腌料刮干净**（不用水冲）—— 残留味噌极易烤焦发黑
3. 鱼皮朝下放烤盘，间距2-3cm
4. 烤之前提前15-20分钟从冰箱取出回温
5. 中层烤 **18-22分钟**（2.5cm厚），温度计确认内部达63°C
6. 取出刷预留腌料，开上火炙烤模式 **1-2分钟**至表面微焦金黄
7. 静置2-3分钟

**2b. 空气炸锅版**：200°C / 400°F
1. 空气炸锅预热200°C（3-4分钟）
2. **同样必须刮掉多余腌料**（空炸离热源更近，味噌更容易焦）
3. 炸篮喷薄油或铺剪好的烘焙纸，鱼**皮朝下**放入，两块之间留间距
4. 烤之前提前15-20分钟从冰箱取出回温
5. **200°C 炸8分钟**（2.5cm厚），不翻面，温度计确认内部达63°C
6. 如果厚度超过2.5cm，延长至10分钟
7. 取出刷预留腌料，**静置5分钟**（空炸锅温度集中，静置让内部熟度均匀很重要）

关键技巧：
- **必须刮掉腌料再烤** —— 味噌含糖和蛋白质极易焦黑，刮掉才能烤出漂亮金黄色
- 选白味噌，黄/赤味噌偏咸偏重需减量
- 带皮烤，鱼皮朝下防止鱼肉过干，还能烤脆
- 全熟口感补救：选脂肪多的三文鱼部位（腹部），出炉淋少许芝麻油
- 空炸版比烤箱快一半（8分钟 vs 18-22分钟），焦化效果更好，但更容易过熟，宁可少1分钟用温度计确认

参考来源：[Just One Cookbook](https://www.justonecookbook.com/miso-salmon/)、[RecipeTin Eats](https://www.recipetineats.com/miso-marinated-side-of-salmon/)、[Tiffy Cooks 空炸版](https://tiffycooks.com/miso-glazed-salmon-air-fryer-recipe/)、[Mikha Eats 空炸版](https://mikhaeats.com/air-fryer-miso-ginger-salmon/)、[下厨房](https://m.xiachufang.com/recipe/102237711/)、[FDA孕期鱼类安全](https://www.fda.gov/food/people-risk-foodborne-illness/meat-poultry-seafood-food-safety-moms-be)

### 照烧银鳕鱼

⚠️ 孕期：银鳕鱼属低汞鱼类，可放心食用。必须全熟，中心温度 ≥ 63°C/145°F

食材（2人份）：
- 银鳕鱼 2块（每块170-200g，约2.5cm厚）
- 盐、黑胡椒少许（腌制用）
- 油 1汤匙/15ml（煎鱼用）

照烧汁：
| 调料 | 用量 | 备注 |
|------|------|------|
| 酱油（生抽） | 3汤匙（45ml） | 日式酱油或普通生抽 |
| 味淋 | 3汤匙（45ml） | 提供甜味和光泽 |
| 清酒或料酒 | 3汤匙（45ml） | 增加风味层次 |
| 白砂糖 | 1.5汤匙（20g） | 帮助收汁挂糊焦糖化 |

经典照烧比例：酱油:清酒:味淋 = 1:1:1 + 半份糖

**1. 调照烧汁**
1. 小锅倒入全部调料，中火加热搅拌至糖溶解，煮沸转小火
2. 继续煮8-10分钟，至浓缩约1/3，能挂住勺背（冷却后会更浓稠）
3. 分两份：一份烤制刷面，一份最后浇淋

**2. 煎鱼（先煎后烤法）**
1. 银鳕鱼吸干水分（**关键，决定能不能煎出焦壳**），两面撒盐和黑胡椒，室温静置10分钟
2. 烤箱预热200°C/400°F
3. 可入烤箱的煎锅（铸铁锅最佳）中火加热，倒入15ml油
4. 油热微冒烟，鱼**皮面朝下**煎2-3分钟至金黄焦脆，翻面再煎2分钟

**3. 进烤箱**：200°C，10-15分钟
1. 鱼表面刷一层照烧汁
2. 连锅入烤箱，200°C烤10-15分钟
3. 中间约7分钟再刷一次照烧汁
4. **温度计确认中心达63°C/145°F**，鱼肉完全不透明

**4. 焦糖化（可选）**
1. 烤好后开上火炙烤模式，距热源15cm
2. 刷最后一层照烧汁，炙烤1-2分钟至表面起泡焦糖化
3. **盯紧！照烧汁含糖高，极易烧焦**

出锅淋剩余照烧汁，撒白芝麻/葱花

关键技巧：
- 银鳕鱼脂肪含量极高（15-20%），天然比普通鳕鱼难干
- 145°F到了就出炉，不要多烤
- 酱汁分次刷形成保护膜锁住水分
- 不要用普通大西洋鳕鱼替代，口感差异巨大

**纯煎备选**（不用烤箱）：中小火煎皮面3分钟+翻面3分钟 → 倒照烧汁盖盖焖2-3分钟 → 开盖收汁翻裹

参考来源：[Just One Cookbook](https://www.justonecookbook.com/teriyaki-sauce/)、[Fish and Family](https://www.fishandfamily.com/post/teriyaki-glazed-black-cod-tips)、[下厨房](https://m.xiachufang.com/recipe/107046924/)

### 照烧银鳕鱼

食材（2-3人份）：
- 银鳕鱼 2块（约300g，提前冰箱解冻）
- 姜片 3-4片
- 食用油 1汤匙（15ml）
- 白芝麻少许（装盘用，可省）

照烧汁（日式经典比例 2:2:2:1）：
- 生抽 2汤匙（30ml）
- 日式料理酒 2汤匙（30ml）
- 味淋 2汤匙（30ml）
- 白糖 1汤匙（12g）

1. 银鳕鱼解冻后厨房纸**彻底吸干水分**（带水煎不上色还溅油）
2. 鱼身两面轻划2-3刀（利于入味，别太深）
3. 照烧汁提前调好搅匀
4. 平底锅中小火热油，放姜片煸香，放鱼块煎2-3分钟至底面金黄
5. 小心翻面再煎2分钟（银鳕鱼嫩，**只翻一次**，多翻会碎）
6. 倒入照烧汁，转小火，用勺子不断将汁舀起淋到鱼上
7. 收汁到浓稠挂在鱼身上即可（约2-3分钟）
8. 装盘撒白芝麻

关键技巧：
- 吸干水分是第一步，水分没擦干什么都白搭
- 只翻一次面，银鳕鱼太嫩多翻必碎
- 收汁时小火，不停淋汁让鱼均匀上色
- 照烧汁里的糖容易焦，火不能大

参考来源：[Just One Cookbook](https://www.justonecookbook.com/teriyaki-sauce/)、[下厨房](https://m.xiachufang.com/recipe/106802498/)、[Chopstick Chronicles](https://www.chopstickchronicles.com/basic-teriyaki-sauce/)

### 大虾烧白菜（鲁菜）

食材（2-3人份）：

主料：
- 大白菜 500g（半棵，黄心最佳）
- 大虾 300g（约10-12只）

配料：
- 大葱 1根（切斜段）
- 姜 1小块（约15g，切片）
- 蒜 3瓣（约10g，切片）

调味：
- 花生油 30ml（2汤匙，花生油更正宗）
- 盐 3g（分次放）
- 黄酒 15ml（1汤匙）
- 蚝油 15ml（1汤匙）
- 生抽 10ml（2茶匙）
- 0卡糖 1/4茶匙
- 玉米淀粉 5g + 水 30ml（勾芡用）

1. **处理白菜**：帮和叶分开，帮子**斜刀切薄片**（约0.5cm，增大入味面积），叶子手撕大块
2. **处理虾**：虾头虾身分离，挑虾线，虾头**对半剪开**（方便煎出虾膏）。虾身用少许盐+黄酒5ml抓匀腌5分钟
3. **炒白菜出水**：热锅倒10ml油，大火先下白菜帮炒1分钟，再下白菜叶+盐1g，大火炒至出水帮子微透明（约2分钟），盛出沥干备用
4. **煎虾头出红油（灵魂步骤）**：不洗锅倒入20ml油，**中小火**放入虾头，用锅铲轻轻按压让虾膏流出，持续煎**3-4分钟**直到油变橘红色虾头焦酥（煎不透会腥，煎透了才是鲜）
5. **煎虾身**：虾头推到一边放入虾身，中火煎至两面变红（每面约1分钟）
6. **爆香料**：放入葱段姜片蒜片翻炒30秒，沿锅边淋入黄酒10ml激香
7. **合体收汁**：加蚝油1汤匙+生抽2茶匙+0卡糖1/4茶匙+清水约100ml，白菜倒回锅中，大火翻炒2-3分钟让白菜吸满虾汁
8. 淋入水淀粉（5g+30ml水）勾芡翻匀出锅

关键技巧：
- **虾头煎透出红油是成败关键**，判断标准：虾膏完全沙化+油变橘红+闻起来纯鲜香不腥
- 白菜先炒出水再回锅，才能吸虾汁而不是出一堆水
- 白菜帮斜刀切片增大入味面积，口感也更好
- 花生油比普通油更香（青岛传统做法）

### 香煎龙利鱼（通用鱼排做法）

食材（2人份）：
- 龙利鱼柳 300g（买带皮的是真龙利鱼）
- 玉米淀粉 15g（约1汤匙，裹面用，煎出来外酥内嫩）
- 料酒 15ml（1汤匙）
- 生抽 15ml（1汤匙）
- 黑胡椒粉 少许（约1g）
- 牛油果油或花生油 15ml（高温煎用）

做法：
1. 龙利鱼**完全化冻**后洗净，用厨房纸**反复按压吸干水分**，正反面都要擦（关键中的关键！水分不除干净，一是腌不进味，二是下锅会散、会溅油、会出大量水变成煮鱼）
2. 切成约3cm宽的块（不要太小，煎的时候会缩）
3. 碗中加料酒15ml、生抽15ml、黑胡椒粉拌匀，腌制**10-15分钟**。⚠️ 严格不超过15分钟，龙利鱼肉质疏松极易吸味，腌久必咸
4. 腌好后倒掉多余汁水，撒入玉米淀粉15g，轻轻翻拌让每块鱼表面均匀裹上薄薄一层（淀粉能锁住水分+形成脆壳）
5. 平底锅中火预热，倒油15ml，油温热到微微冒烟时放入鱼块。**放入后不要动**，煎2-3分钟让底面形成金黄硬壳后再翻面（提前翻=散+粘锅）
6. 翻面后再煎2-3分钟，两面金黄、鱼肉全熟（中心不透明白色）即出锅
7. 出锅前可撒少许白芝麻增香（可选）

关键技巧与避坑：
- **防散三要素**：①吸干水分 ②裹淀粉 ③下锅后不急翻
- **防粘锅**：锅要充分预热再放油，油热再放鱼；用不粘锅最保险
- 煎的过程中鱼会出水，这是正常的，继续煎水分会被收掉蒸发掉
- 不要切太小，煎完会缩约20-30%

### 香煎整鱼

适用鱼种：金鲳鱼（首选，刺少肉嫩）、黄花鱼、鲈鱼、鲫鱼、多宝鱼等。根据鱼大小调整煎制时间。

食材（2人份）：
- 整鱼 1-2条（共约500g，让鱼店去鳞去内脏）
- 姜 3-4片（腌制用）+ 姜丝少许（煎时用）
- 葱 1根（葱白切段塞鱼肚，葱绿切葱花出锅撒）
- 蒜 2-3瓣（切片）
- 料酒 15ml（1汤匙）
- 盐 3g（约半茶匙，抹鱼身用）
- 生抽 15ml（1汤匙，出锅前调味）
- 玉米淀粉 适量（薄薄抹一层，锁水+煎出脆皮）
- 牛油果油或花生油 20-30ml（煎鱼要多一点油）
- 柠檬 半个（出锅挤汁，去腥提鲜，可选）

做法：
1. 鱼洗净，撕掉肚内黑膜（腥味来源），正反面各斜划2-3刀（深到鱼骨，入味+受热均匀）
2. 鱼身正反面均匀抹盐3g，鱼肚里塞入姜片和葱白段，淋料酒15ml，腌制10分钟翻面再腌10分钟（共20分钟）
3. 腌好后用厨房纸把鱼表面水分彻底吸干（关键！带水煎=溅油+粘锅+不脆），薄薄抹一层玉米淀粉
4. 不粘锅倒油20-30ml，中火烧至油面微微冒烟，轻轻放入鱼。放入后不要动，煎2-3分钟让底面定型金黄后再翻面（提前翻必粘必散）
5. 翻面再煎2-3分钟至两面金黄微焦
6. 放入姜丝、蒜片，翻炒几秒爆出香味，沿锅边淋入生抽15ml
7. 鱼肚内部全熟即可出锅，撒葱花，挤几滴柠檬汁

关键技巧：
- 防粘三要素：①鱼表面擦干 ②锅和油要够热 ③下锅后不急翻，定型2分钟再动
- 用不粘锅最保险；铁锅也可以但要锅烧更热、全程盯着
- 鱼身划刀要够深（到鱼骨），否则中间不熟
- 腌过有底味了，生抽适量即可，不额外加盐
- 油可以稍多，煎鱼不是炒菜
- 大鱼（鲈鱼等）每面煎3-4分钟，小鱼（鲳鱼等）每面2-3分钟

- 小红书 @皮皮椰（7k收藏） — 抹淀粉锁水+腌制20分钟
- 小红书 @一枚家庭煮夫（4k收藏） — 柠檬提鲜

### 薄荷煎鱿鱼（2人份，约10分钟）

主料：
- 小鱿鱼 400g（整只小鱿鱼，不是鱿鱼圈）

酱料（提前混合）：
- 新鲜薄荷叶 15g，切碎
- 蒜 4瓣（20g），切末
- 橄榄油浸凤尾鱼 4-5条（约30g，含油一起捣碎）
- 三色小番茄 100g，对半切
- 橄榄油 30ml
- 生抽 1茶匙
- 0卡糖 半茶匙

做法：
1. 鱿鱼去内脏、去软骨、撕掉外膜，洗净后**彻底擦干**（水分是出水的元凶）
2. 铸铁锅大火烧到冒烟，**不放油**，鱿鱼直接下锅，每面煎2-3分钟至焦香上色
3. 倒入混合好的酱料，盖盖焖1分钟
4. 开盖翻拌均匀出锅

关键技巧：
- **小鱿鱼+彻底擦干+锅烧到冒烟+不放油** = 不出水的四个条件
- 大鱿鱼容易出水，必须用小鱿鱼
- 凤尾鱼是鲜味和油脂来源，不能省

参考来源：小红书 @soooo（1690赞）、@徐抽（687赞）、@爱自由的luffy

### 黄金炸带鱼（新荣记复刻版）

⚠️ 需提前准备：盐水腌70分钟 + 冰箱风干过夜（至少2小时）
⚠️ 孕期：油炸食品一周最多安排一次

食材：
- 新鲜带鱼 1000g（必须鲜带鱼，冻带鱼做不出金黄色）

盐水：水1000ml + 盐18g + 花雕酒25ml + 姜几片 + 葱段

炸油：约1-1.5升

做法：
1. **刮银膜（核心步骤）**：用刀背把带鱼表面银色膜刮干净（不刮=灰色不是金黄色）
2. 带鱼切段，放入盐水腌**70分钟**
3. 捞出沥干，放冰箱**风干过夜**（至少2小时，越干越脆）
4. **裸炸不裹粉**：油温200°C，带鱼下锅**不要翻面**，炸至金黄浮起（约3-4分钟）
5. 可选复炸：油温升至210°C，回锅炸15-20秒更酥脆

关键技巧：
- **必须刮银膜**，否则颜色灰暗不金黄
- **必须鲜带鱼**，冻的水分太多炸不脆
- **裸炸不裹任何粉**，新荣记做法
- **不翻面**，翻了容易碎
- 需提前一天准备，当天临时做不了
