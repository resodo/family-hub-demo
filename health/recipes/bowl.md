### 藜麦基础煮法

**电饭锅版（推荐，最简单）**（1杯藜麦 ≈ 170g → 出品约3杯）：

食材：藜麦1杯（170g）、水约305ml（藜麦:水 = 1:1.8）、盐1/2茶匙

1. **必须洗**：细筛网流水冲洗藜麦2-3遍，搓洗到水不再起泡，去皂苷（不洗会苦）。可提前泡10分钟更好
2. 沥干放入电饭锅，加水和盐
3. 按"煮饭"键，正常煮
4. **跳闸后再焖10分钟**（关键！不要马上开盖）
5. 用叉子轻轻拨松，颗粒分明即成

> 水量比例：1:1.8（标准），想更软用1:2

**灶台版**（1杯藜麦 ≈ 170g → 出品约3杯）：

食材：藜麦1杯（170g）、水约300ml（1:1.75）、盐1/2茶匙

1. 冲洗同上
2. **干炒提香**（可选）：洗净沥干倒入干锅，中火翻炒2-3分钟至闻到坚果香
3. 加水和盐，大火烧开，转最小火盖盖煮15分钟，不掀盖
4. **关火焖10分钟**
5. 用叉子轻轻拨松

判断是否熟：藜麦变透明，能看到白色小卷须（胚芽）脱出。中间有白色硬芯=没熟

保存：冷藏密封5-7天，冷冻1-2个月

**藜麦+大米混煮**（电饭锅）：

藜麦占总米量20-30%，总水量比纯白米多约10%

| 大米：藜麦 | 说明 |
|-----------|------|
| 3:1（入门） | 口感接近纯米饭，如200g大米+60g藜麦 |
| 2:1（常用） | 能明显吃到藜麦颗粒，营养提升 |
| 1:1 | 藜麦感强，习惯后用 |

1. 大米正常淘洗，藜麦细筛网冲洗沥干
2. 混合放入电饭锅，水比纯白米**多10%**
3. "煮饭"或"杂粮"模式
4. 煮好焖5-10分钟再开盖搅拌

注意：白米/茉莉香米效果最好。糙米烹煮时间比藜麦长，不适合直接混煮

参考来源：[The Kitchn](https://www.thekitchn.com/how-to-cook-quinoa-cooking-lessons-from-the-kitchn-63344)、[Inspired Taste](https://www.inspiredtaste.net/38095/how-to-cook-quinoa/)、[下厨房](https://m.xiachufang.com/recipe/103456822/)、[知乎](https://zhuanlan.zhihu.com/p/52832520)

### 烤西兰花

食材：西兰花1头（400-500g）、橄榄油2汤匙（30ml）、蒜3-4瓣切碎、盐3/4茶匙、黑胡椒适量、柠檬汁少许（可选，出锅挤）

预处理（通用）：
1. 西兰花切均匀小朵（小朵更容易焦脆）
2. **洗净后彻底擦干/沥干** —— 脆边的关键，有水=蒸不是烤
3. 大碗里加橄榄油、蒜碎、盐、黑胡椒，翻拌均匀，每朵都裹上油

**烤箱版**：220°C，18-22分钟
1. 烤箱预热220°C，烤盘铺烘焙纸
2. 西兰花**单层平铺不堆叠**（堆叠=蒸=软塌，铺不下就分两盘）
3. 中层烤18-22分钟，中途10分钟翻面一次
4. 边缘金棕色、茎部叉子能戳透即出炉

**空气炸锅版**：190°C，8-10分钟
1. 预热190°C（3分钟）
2. 单层摆放不堆叠，分批炸效果更好
3. 190°C 烤8-10分钟，中途翻动一次
4. 想更脆：最后1分钟升温至200°C

脆边秘诀：
- **沥干水分**最重要，水分是脆边大敌
- 油要裹匀，不够油会干焦
- 单层平铺不堆叠
- 切小均匀，大小一致确保均匀受热

参考来源：[Natasha's Kitchen](https://natashaskitchen.com/roasted-broccoli/)、[Downshiftology](https://downshiftology.com/recipes/roasted-broccoli/)、[知乎](https://zhuanlan.zhihu.com/p/356836137)、[下厨房](https://m.xiachufang.com/recipe/106856206/)

### 烤南瓜

食材（2-3人份）：贝贝南瓜/日本南瓜1个（约700g-1kg）、橄榄油1-2汤匙（15-30ml）、盐1/2茶匙、黑胡椒适量

可选调味风格：
- **原味**：只用盐+黑胡椒（推荐搭配照烧鱼碗）
- **香料版**：加蒜粉1/2茶匙+烟熏辣椒粉1/4茶匙
- **甜味版**：加枫糖浆1汤匙+肉桂粉1/4茶匙

切南瓜技巧（皮很硬，安全第一）：
1. **微波预软化**（推荐）：整颗南瓜微波高火2-3分钟，外皮稍软后大幅降低切割难度
2. 从顶部纵向切两半，挖去籽和内瓤
3. 切成2.5-4cm厚的月牙形楔块
4. **皮不用削** —— 贝贝南瓜的皮烤熟后完全可以吃
5. 大碗中淋油撒盐黑胡椒，翻拌均匀

**烤箱版**：220°C/425°F，25-30分钟
1. 烤盘铺烘焙纸，南瓜块单层平铺切面朝下，留间距
2. 烤12-15分钟后翻面，再烤10-15分钟至金黄酥软
3. 叉子能轻松插入即可

**空气炸锅版**：200°C/400°F，12-16分钟
1. 预热200°C，南瓜块单层放入炸篮
2. 炸7-8分钟翻面，再炸5-8分钟至外金内软
3. 小块（2cm以下）总时间可缩短到10-12分钟

参考来源：[Love and Lemons](https://www.loveandlemons.com/roasted-kabocha-squash/)、[Wholesome Yum](https://www.wholesomeyum.com/kabocha-squash-recipe/)、[下厨房](https://m.xiachufang.com/recipe/103924961/)

### 味噌三文鱼碗（空气炸锅版）

便当组装指南（1人份小鱼便当）

**组成：** 藜麦 + 味噌烤三文鱼 + 烤南瓜 + 煮毛豆 + 牛油果 + 煮蛋

食材清单：
| 组件 | 食材 | 用量 | 菜谱位置 |
|------|------|------|---------|
| 底层 | 藜麦（提前煮好） | 150-180g（熟重） | 本文件「藜麦基础煮法」 |
| 蛋白质 | 味噌烤三文鱼 | 1块（170g生重） | seafood.md「味噌烤三文鱼」空炸版 |
| 蔬菜1 | 烤南瓜 | 150g（切块） | 本文件「烤南瓜」空炸版 |
| 蔬菜2 | 煮毛豆 | 80g（去壳重） | 下方「煮毛豆」 |
| 好脂肪 | 牛油果 | 半个 | 切块+淋柠檬汁 |
| 蛋 | 煮蛋 | 1个 | 对半切 |

**前一晚准备：**
1. 三文鱼腌味噌腌料（30分钟-3小时均可，不超过3小时），冷藏过夜不腌
2. 藜麦煮好冷藏（可一次煮3天的量）

**当天早上流程（阿姨操作，约25分钟）：**
1. 煮蛋：冷水下锅，水开后中火煮8分钟 → 捞出冰水浸泡 → 剥壳对半切
2. 煮毛豆：水烧开加盐，毛豆（带壳）煮5分钟 → 捞出过凉水 → 剥壳备用
3. 空炸南瓜：南瓜切2.5cm块，拌橄榄油+盐 → 200°C炸12-16分钟（中途翻面）
4. 空炸三文鱼：三文鱼从冰箱取出回温15分钟，**刮掉多余腌料** → 200°C炸8分钟（南瓜炸完紧接着炸）
5. 三文鱼静置5分钟，期间装碗

**装碗顺序（便当盒）：**
1. 藜麦铺底（微波加热1分钟或室温即可）
2. 烤南瓜、煮毛豆、煮蛋摆放
3. **牛油果单独小盒装**，淋柠檬汁防氧化，吃前再放（和热菜一起装会变色变软）
4. 三文鱼放最上面（保持表面焦化口感）
5. **酱汁单独小容器**：味噌酱汁（预留腌料1汤匙 + 少许温水调稀）或直接不加酱（三文鱼本身已有味噌味）

**营养亮点：** 欧米伽3（三文鱼）+ 完整蛋白质+铁+叶酸（藜麦）+ 维A（南瓜）+ 叶酸+健康脂肪（牛油果）+ 胆碱（蛋）+ 植物蛋白（毛豆）

### 煮毛豆（碗用基础组件）

食材：带壳毛豆 200g（出仁约80g）、盐 1茶匙

做法：
1. 毛豆洗净，剪去两头尖角（可选，方便入味）
2. 水烧开加盐，下毛豆大火煮5分钟
3. 捞出立刻过冰水（保持翠绿色）
4. 沥干，剥壳备用

保存：剥壳后冷藏密封3天，冷冻1个月

### 柠檬香草烤鸡腿（空气炸锅版）

食材（4-5个鸡腿）：
- 带骨带皮鸡腿 4-5个（约800g）

腌料：
- 橄榄油 2汤匙（30ml）
- 新鲜柠檬汁 3汤匙（约1.5个柠檬）
- 柠檬皮屑 1个柠檬的量
- 蒜末 3-4瓣（约12g）
- 新鲜迷迭香 1汤匙切碎
- 新鲜百里香 1汤匙切碎
- 甜椒粉 1茶匙
- 洋葱粉 1/2茶匙
- 粗盐 1茶匙
- 黑胡椒 1茶匙

做法：
1. 鸡腿厨房纸**彻底拍干**（皮脆的关键）
2. 混合所有腌料，鸡腿放密封袋倒入腌料揉匀裹满，**冷藏至少1小时，过夜最佳**
3. 鸡腿皮朝上放入炸篮（喷少量油防粘），**200°C炸18-22分钟**，中途翻面一次
4. 温度计插入最厚处**必须≥74°C**（孕期安全）。大鸡腿可能需22-25分钟
5. 休息2-3分钟后切块装碗

碗用法：切块或撕丝，配藜麦+烤蔬菜+牛油果+酱汁。冷藏保存5天，复热空气炸锅180°C 3-4分钟恢复脆皮

关键技巧：
- **柠檬皮比柠檬汁更香**，一定要刮皮屑
- 腌制1小时+才够味，但不超24小时（酸会让肉质变粉）
- **皮朝上**让热风直吹皮面才脆
- 备餐友好：一次做5个分装，配不同碗底和酱汁吃一周

- [Cooks Well With Others - 柠檬香草空气炸锅鸡腿](https://cookswellwithothers.com/2022/07/19/air-fryer-lemon-herb-chicken-thighs/)
- [AirFried.com - 柠檬蒜香鸡腿](https://airfried.com/air-fryer-lemon-garlic-chicken-thighs/)

### 韩式烤牛肉（空气炸锅版）

食材（4人份）：
- 牛里脊/牛肩肉 680g，逆纹切3mm薄片（冷冻15-20分钟更好切）
- 葱 3根（切段）
- 胡萝卜丝 80g

腌料：
- 酱油 60ml（4汤匙）
- 红糖/白糖 30g（2汤匙）
- 芝麻油 15ml（1汤匙）
- 蒜末 3瓣（约10g）
- 姜末 1茶匙（约5g）
- 黑胡椒 适量
- 熟白芝麻 2汤匙
- 亚洲梨半个磨泥（可选，嫩肉）

做法：
1. 牛肉+葱段+胡萝卜丝放密封袋，倒入腌料揉匀，冷藏至少30分钟（过夜更佳，最长24小时）
2. 取出**沥掉多余腌汁**（重要！否则烧焦）
3. 单层铺入炸篮不堆叠，**200°C炸12分钟**，中途翻面一次
4. 分批炸，确保空气循环才有焦化脆边

关键技巧：
- **切薄3mm**才能入味+快速焦化
- **沥掉多余腌汁**防底部积水变蒸
- **不堆叠**单层铺开才有脆边
- 出锅前最后1-2分钟可升温210°C帮助上色
