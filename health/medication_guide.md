# 孕期用药安全速查

> 本指南参考：ACOG（美国妇产科学会）、MotherToBaby、丁香医生用药助手
> 任何用药前建议咨询产科医生确认

## 快速查询表

### ✅ 孕期可用（常见场景）

| 症状 | 首选药物 | 用法 | 说明 |
|------|---------|------|------|
| 发烧/头痛 | **对乙酰氨基酚** | 500mg，q6h，日最大2g | 全孕期安全首选 |
| 过敏/荨麻疹 | **氯雷他定** | 10mg/日 | B类，优先选择 |
| 鼻塞/鼻炎 | **布地奈德鼻喷** | 每鼻孔1-2喷/日 | B类，局部用药安全 |
| 鼻腔干燥 | **生理盐水冲洗** | 随需 | 完全安全 |
| 恶心/孕吐 | **维生素B6** | 25mg，tid | ACOG一线推荐 |
| 孕吐加重 | **B6+多西拉敏** | B6 25mg + 多西拉敏 12.5mg | ACOG阶梯治疗 |
| 便秘 | **乳果糖** | 15-30ml/日 | 不吸收，安全 |
| 胃灼热 | **碳酸钙咀嚼片** | 随需 | 同时补钙 |
| 腹泻 | **蒙脱石散** + 补液盐 | 按说明 | 物理吸附，不吸收 |

### ❌ 孕期禁用

| 药物 | 风险 |
|------|------|
| **布洛芬/阿司匹林** | 孕晚期致动脉导管早闭，全孕期避免 |
| **四环素/多西环素** | 牙齿染色、骨骼影响 |
| **甲氧氯普胺(胃复安)** 长期 | 锥体外系反应 |
| **含碘喉片** | 影响胎儿甲状腺 |
| **维A酸类（口服）** | 致畸 |

### ⚠️ 需医生评估

| 药物 | 说明 |
|------|------|
| 抗生素 | 头孢类、阿奇霉素相对安全，但需医生处方 |
| 中成药 | 成分复杂，默认不推荐，个别经典方除外 |
| 止咳药 | 右美沙芬短期可用，含可待因的禁用 |

## 小鱼专项注意

- **鼻炎**：布地奈德鼻喷 + 生理盐水冲洗为主，避免口服减充血剂（伪麻黄碱）
- **补铁副作用**：便秘可加乳果糖，黑便属正常不用担心
- **弓形虫暴露后**：如误食生肉/接触猫粪，立即就医检测IgM

## 糖豆用药注意

- **退烧**：首选布洛芬混悬液（美林），按体重计算剂量（13kg → 约4ml）
- **过敏应急**：西替利嗪滴剂 2.5mg
- **禁用**：阿司匹林（Reye综合征风险）、含蜂蜜止咳糖浆（<3岁）
- **注意蛋成分**：部分药物辅料含卵磷脂/蛋白，用药前看说明书
